## 2026 plaan

### Peenrad

```
P1: Maasikad + küüslauk + kurgirohi
P2: Maavitsalised (tomat, tšilli, füüsal)
P3: Liblikõielised + juurviljad (herned, oad, porgand, sibul, porru)
P4: Kõrvitsalised (kurk, suvikõrvits, patisson)
```

### Kasvuhoone

- Kurk 'Suyu Long'
- Tomatid ('Malle', 'Pille', 'San Marzano 2')
- Arbuus 'Sugar Baby'

### Eraldi ala

- Kartul 'Solist' (jaanipäevaks)

### Viljapuud

| Kultuur | Sordid | Märkus |
|---------|--------|--------|
| **Ploomid** | 'Emma Leppermann', 'Liisu' | 'Emma Leppermann' tolmeldab 'Liisu' |
| **Sammasõunapuud** | 'Baiba', 'Inese' | Läti sordid, talvekindlad |
