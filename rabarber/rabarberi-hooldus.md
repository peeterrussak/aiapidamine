# Rabarberi kasvatamine

## Ülevaade

Rabarber (*Rheum rhabarbarum*) on **püsikkultuur**, mis võib kasvada samas kohas 10–20 aastat! Üks esimesi kevadsaake aias – varred on valmis juba mais.

**Miks kasvatada rabarbert:**
- **Püsik:** Istuta üks kord, korja aastaid
- **Varajane saak:** Mai–juuni (üks esimesi!)
- **Hooldusevaba:** Peaaegu ei vaja hooldust
- **Mitmekülgne:** Koogid, moosid, kompotid, mahlad
- **Dekoratiivne:** Suured lehed, punased varred

**⚠️ Oluline:** Söödavad on AINULT VARRED! Lehed sisaldavad mürgist oksaalhapet.

## Kasvatamine

### Koha valik

| Tegur | Nõue |
|-------|------|
| Valgus | Päike kuni poolvari (min 4–6 tundi päikest) |
| Muld | Rikkas, sügav, hea drenaažiga |
| Niiskus | Niiskust hoidev, aga mitte vesine |
| Ruum | 1 × 1 m taime kohta (kasvab suureks!) |

**Koha eripärad:**
- Rabarber kasvab ka poolvarjus (vähem saagikas)
- Vali **püsiv koht** – taim jääb sinna 10–20 aastaks!
- Sobib hästi peenra äärde või eraldi alale
- Võib kasutada dekoratiivtaimena (suured lehed)

### Istutamine

**Millal istutada:**
- **Kevad:** Märts–aprill (enne kasvu algust)
- **Sügis:** September–oktoober (enne külmi)

**Istutusmaterjal:**

| Viis | Kirjeldus | Saak algab |
|------|-----------|------------|
| **Juuretükk (jaotamine)** ⭐ | Vanema taime juurestiku tükk 1–2 pungaga | 2. aastal |
| **Potitaim** | Ostetud taim | 2. aastal |
| **Seemnest** | Võimalik, aga aeglane | 3.–4. aastal |

**Kuidas istutada:**

1. Kaeva auk 40–50 cm sügav ja lai
2. Sega mulda küpse komposti või sõnnikuga
3. Aseta juurestik nii, et pungad on **5 cm mullapinna all**
4. Tihenda muld ja kasta põhjalikult
5. Multši 5–10 cm paksuse kihiga

```
ISTUTAMISE SKEEM:
         ┌─────────────────────────┐
         │     Mullapind           │
    ═════╪═════════════════════════╪═════
         │  ↑ 5 cm                 │
         │  ● Pung                 │
         │  │                      │
         │  ╰──┬──╯                │
         │     │  Juurestik        │
         │  ╭──┴──╮                │
         │  │█████│                │
         │  ╰─────╯                │
         │     │                   │
         │   Juured                │
         └─────────────────────────┘
```

**⚠️ Oluline:** Ära istuta liiga sügavale ega liiga madalale! Pungad peavad olema ~5 cm mulla all.

## Hooldus

### Aastaringne hooldus

| Kuu | Tegevus |
|-----|---------|
| Märts | Eemalda talvekate, lisa komposti |
| Aprill | Kasvu algus, jälgi niiskust |
| Mai–juuni | **Korjamine** (peamine hooaeg) |
| Juuni lõpp | **Lõpeta korjamine** (jaanipäeva paiku) |
| Juuli–august | Lase taimel taastuda, kasta kuival ajal |
| September | Lisa komposti, võib jagada taimi |
| Oktoober–november | Multši talveks (valikuline) |

### Väetamine

| Aeg | Tegevus |
|-----|---------|
| Kevad (märts–aprill) | Lisa 5–10 cm komposti või kõdunenud sõnnikut taime ümber |
| Pärast korjamist (juuli) | Lisa komposti, et taim taastuks |

**NB:** Rabarber on "näljane" taim – armastab rikast mulda ja regulaarset kompostilisandust!

### Kastmine

- Kasta kuival perioodil (eriti korjamise ajal)
- Hoia muld ühtlaselt niiske
- Multš aitab niiskust hoida
- Põud = peenemad varred

### Õisvarte eemaldamine

**Kui rabarber hakkab õitsema:**
- Lõika õievarred kohe maha!
- Õitsemine nõrgestab taime
- Varred jäävad peenemaks

```
ÕIEVARS:
    🌸 ← Lõika see maha!
     │
     │
  ╱     ╲
 ╱       ╲  ← Lehed jäävad alles
```

### Multšimine

| Hooaeg | Materjal | Eesmärk |
|--------|----------|---------|
| Kevad | Kompost | Toitained |
| Suvi | Hein, põhk | Niiskuse hoidmine |
| Talv | Lehed, põhk | Kaitse külma eest (valikuline) |

## Korjamine

### Millal korjata

| Aeg | Juhis |
|-----|-------|
| **Algus** | Mai (kui varred 25–30 cm pikad) |
| **Põhihooaeg** | Mai–juuni |
| **Lõpp** | Jaanipäev (24. juuni) |

**⚠️ Oluline:** Lõpeta korjamine jaanipäeva paiku! Taim vajab suvel aega taastumiseks ja jõu kogumiseks talveks.

### Kuidas korjata

**Õige viis:**
1. Haara varrest tüve lähedalt
2. **Keera ja tõmba** (mitte lõika!)
3. Vars tuleb puhtalt välja

**Vale viis:**
- ❌ Ära lõika noaga (jätab kännu, mis mädaneb)
- ❌ Ära võta üle 1/3 vartest korraga
- ❌ Ära korja esimesel aastal üldse!

```
ÕIGE KORJAMINE:
      ╱╲
     ╱  ╲  ← Leht (viska ära!)
    ╱    ╲
   ╱      ╲
   │      │
   │ VARS │ ← Söödav osa
   │      │
   ╰──────╯
     ↑
   Keera ja tõmba siit
```

### Korjamise reeglid

| Reegel | Selgitus |
|--------|----------|
| **1. aasta:** Ära korja üldse | Lase taimel juurduda |
| **2. aasta:** Korja kergelt | Võta vaid mõned varred |
| **3.+ aasta:** Täissaak | Korja regulaarselt, max 1/3 korraga |
| **Lõpeta jaanipäevaks** | Taim vajab taastumisaega |

### Saagi ootus

| Taime vanus | Saak |
|-------------|------|
| 1. aasta | 0 (ära korja!) |
| 2. aasta | 0,5–1 kg |
| 3.+ aasta | 2–5 kg taime kohta |

## Säilitamine ja kasutamine

### Värskelt

- Külmkapis: 1–2 nädalat (kilekotis)
- Keera lehed kohe peale korjamist ära (võtavad niiskust)

### Külmutamine ⭐

Parim viis rabarberi säilitamiseks!

1. Pese varred
2. Lõika 2–3 cm tükkideks
3. **Ära blanšeeri** (pole vaja!)
4. Paki kilekottidesse
5. Külmuta

**Säilivus:** 8–12 kuud

### Keetmine/küpsetamine

| Kasutusviis | Kirjeldus |
|-------------|-----------|
| **Kompott** | Keeda suhkruga, serveeri külmana |
| **Moos** | Rabarber + suhkur (1:1), keeda pakseks |
| **Kook** | Rabarbrikook, tassikook, crumble |
| **Pirukas** | Klassikaline rabarberipirukas |
| **Mahl** | Pressi või keeda, lisa suhkrut |
| **Hapendamine** | Jah, võimalik! |

### Retseptiidee: Rabarberikompott

```
KOOSTIS:
- 500 g rabarbert (2–3 cm tükid)
- 150–200 g suhkrut (maitse järgi)
- 200 ml vett
- (valikuline: vanillikaun, sidrunikoor)

TEGEMINE:
1. Keeda vesi ja suhkur siirupiks
2. Lisa rabarberitükid
3. Keeda 5–10 min (kuni pehme, aga mitte pudruks)
4. Jahuta, serveeri kohupiima või jäätisega
```

## Taime jagamine (uuendamine)

### Millal jagada

- **Parim aeg:** Varakevad (märts–aprill) enne kasvu algust
- **Alternatiiv:** Sügis (september–oktoober)
- **Kui sageli:** Iga 5–8 aasta järel (kui saak väheneb)

### Kuidas jagada

1. Kaeva kogu taim üles
2. Loputa muld juurtest maha
3. Lõika terava noaga 3–5 tükki
4. Igal tükil peab olema **vähemalt 1–2 punga ja juured**
5. Istuta tükid uutesse kohtadesse
6. Kasta hästi

```
JAGAMINE:
        Vana taim
      ╭───────────╮
      │  ● ● ● ●  │ ← Pungad
      │  ║ ║ ║ ║  │
      │  ╰─┴─┴─╯  │
      │   Juured  │
      ╰───────────╯
           │
           ▼
    ╭───╮     ╭───╮
    │ ● │     │ ● │ ← Uued tükid
    │ ║ │     │ ║ │   (igaühel 1–2 punga)
    │ ╰─╯     │ ╰─╯
    ╰───╯     ╰───╯
```

### Miks jagada?

- Taim muutub vanaks ja vähem saagikaks
- Keskosa "tühjeneb"
- Saad uusi taimi tasuta!
- Värskendab taime elujõudu

## Probleemid ja kahjurid

### Kahjurid

| Kahjur | Tunnused | Tõrje |
|--------|----------|-------|
| **Rabarberimärkakas** | Augud lehtedel | Käsitsi korjamine, ei ole tõsine |
| **Lehetäid** | Kogunevad lehtedele | Tugev veejuga |
| **Teolised** | Näritud noorvõrsed | Õllelõksud, käsitsi |

### Haigused

| Haigus | Tunnused | Ennetamine/tõrje |
|--------|----------|------------------|
| **Kroonmädanik** | Taim närbub, juured mädanevad | Hea drenaaž! Eemalda haige taim |
| **Lehelaik** | Pruunid laigud lehtedel | Eemalda haigeid lehti, hea õhuring |
| **Jahukaste** | Valge kate lehtedel | Hea õhuring, ära kasta lehti |

### Ennetamine

1. **Hea drenaaž** – rabarber ei talu "märgi jalgu"
2. **Õhuring** – ära istuta liiga tihedalt
3. **Puhtus** – korista sügiseks vanad lehed
4. **Terve istutusmaterjal** – kontrolli juurestikku

## Rabarber kaevamisvabas aias

### Paigutus

Rabarber sobib hästi:
- **Peenra äärde** (ei sega rotatsiooni)
- **Eraldi alale** püsikultuuride juurde
- **Marjapõõsaste lähedusse**
- **Dekoratiivsele alale** (suured ilusad lehed!)

```
NÄIDISPAIGUTUS:
┌─────────────────────────────────────────────────┐
│                    AED                          │
│  ┌──────────┐                                   │
│  │ Rabarber │ ← Aia servas, püsiv koht          │
│  │  🌿  🌿  │                                   │
│  │  🌿  🌿  │   (2–4 taime on perele küllalt)   │
│  └──────────┘                                   │
│        │                                        │
│   Marjapõõsad, ürdinurk jne                     │
└─────────────────────────────────────────────────┘
```

### Sobib kokku

| Naaber | Kasu |
|--------|------|
| Marjapõõsad | Sama hooldusrežiim |
| Õunapuud | Kasutab maapinda, varjutaluv |
| Maapirn | Mõlemad püsikud |
| Maitsetaimed (püsikud) | Püsikute ala koos |
