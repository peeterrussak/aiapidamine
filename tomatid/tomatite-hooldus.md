## Ettekasvatamine

### Ajakava

| Tegevus | Aeg | Märkused |
|---------|-----|----------|
| Külv | Märts keskpaik (2.–3. nädal) | 6–8 nädalat enne istutamist |
| Idanemine | 7–14 päeva | 20–25°C, niiske |
| Ümberpotimine | Aprill (2.–3. nädal) | Kui 2–4 päris lehte |
| Karastamine | 1 nädal enne istutamist | Vii päevaks õue, too ööseks sisse |
| Istutamine KH | Mai keskpaik–lõpp | Kui öökülmad möödas |
| Istutamine avamaa | Juuni algus | Pärast öökülmi |

### Idanemistingimused

| Parameeter | Väärtus |
|------------|---------|
| Temperatuur | 20–25°C |
| Niiskus | Ühtlane, mitte märg |
| Valgus | Pärast idanemist PALJU valgust |
| Idanemisaeg | 7–14 päeva |

### Sammud

1. **Külv:** Külva seemned 0,5–1 cm sügavusele niiskesse mulda
2. **Kata:** Kata kilega kuni idanemiseni
3. **Valgus:** Pärast idanemist kohe tugeva valguse kätte
4. **Temperatuur:** Pärast idanemist veidi jahedam (18–20°C) – takistab venitust
5. **Ümberpotimine:** Kui 2–4 päris lehte → 8–10 cm potti, istuta sügavamale (kuni esimeste lehtedeni)
6. **Karastamine:** 1 nädal enne istutamist – vii päevaks õue

## Istutamine

### Kasvuhoonesse

| Parameeter | Väärtus |
|------------|---------|
| Aeg | Mai keskpaik–lõpp |
| Mullatemperatuur | Min 15°C |
| Ridade vahe | 60–80 cm |
| Taimede vahe | 50–60 cm |
| Taimi 3m² kohta | 6–8 tk |

### Avamaale

| Parameeter | Väärtus |
|------------|---------|
| Aeg | Juuni algus (pärast öökülmi!) |
| Mullatemperatuur | Min 15°C |
| Ridade vahe | 60–80 cm |
| Taimede vahe | 50–60 cm |
| Koht | Päikeseline, tuulevari |

### Istutamise sammud

1. Kaeva auk 2× suurem kui pott
2. Lisa kompostisegu auku
3. Eemalda taim potist ettevaatlikult
4. Istuta **sügavamale** kui potis kasvas (varrest kasvavad lisajuured)
5. Kasta korralikult
6. Multši (kui avamaal)
7. Paigalda tugi kohe

## Hooldus

### Tugi ja vormilõikus

**Kõrgekasvulised (kasvuhoone):**
- Vaiad (180–240 cm) VÕI nöör lakke
- Seo taim iga 20–30 cm tagant
- Eemalda KÕIK kõrvalvõsud (kaenlast kasvavad võrsed)
- Kontrolli 2× nädalas

**Madalakasvulised (avamaa):**
- Kerge tugi (60–100 cm vaiad)
- Võib jätta ka ilma toeta (maas roomavad)
- Kõrvalvõsusid ei pea nii rangelt eemaldama

### Kõrvalvõsude eemaldamine

```
KÕRVALVÕSU:
                    ↑ peavarre jätk
                    │
        ┌───────────┼───────────┐
        │           │           │ ← leht
        │     ╲     │     ╱     │
        │      ╲    │    ╱      │
        │       ●───┼───●       │ ← EEMALDA see võrse!
        │      ╱    │    ╲      │    (kaenlast kasvav)
        │     ╱     │     ╲     │
        └───────────┼───────────┘
                    │
                    ↓ varre jätk
```

**Miks eemaldada?**
- Taim suunab energia viljadesse, mitte lehemassisse
- Parem õhuring → vähem haigusi
- Suuremad viljad

### Kastmine

| Faas | Sagedus | Märkused |
|------|---------|----------|
| Istutamisjärgne | Iga päev | Kuni juurdub |
| Kasvuperiood | 2–3× nädalas | Sügavalt, mulda |
| Viljade valmimine | 2–3× nädalas | Ühtlane niiskus! |

**Põhimõtted:**
- Kasta MULDA, mitte lehti (haiguste ennetamine)
- Kasta hommikul
- Ühtlane niiskus – ebaühtlane kastmine põhjustab viljade lõhenemist
- Tilkkastmine on ideaalne

### Väetamine

| Aeg | Väetis | Märkused |
|-----|--------|----------|
| Istutamisel | Kompost auku | 1–2 peotäit |
| Alates õitsemisest | Nõgesevesi 1:10 | Iga 7–10 päeva |
| Viljade valmimine | Kaaliumrikas (puutuhk) | Peotäis taime kohta |

**NB:** Liiga palju lämmastikku = palju lehti, vähe vilju!

### Ladvamine

- **Aeg:** Juuli lõpp – august algus
- **Miks:** Et viimased viljad jõuaksid valmida
- **Kuidas:** Näpista peamise varre tipp ära
- **Jäta:** 4–5 kobarat, mis jõuavad küpseda

## Kahjurid ja haigused

### Haigused

| Haigus | Tunnused | Ennetamine/tõrje |
|--------|----------|------------------|
| **Lehemädanik** (Phytophthora) | Pruunid laigud lehtedel ja viljadel | Hea õhuring, ära kasta lehti, vasepreparaat |
| **Hallhallitus** (Botrytis) | Hall kirme viljadel | Hea õhuring, eemalda haigeid vilju |
| **Verticillium** | Pooleldi närbuvad taimed | Külvikord, terved seemned |
| **Lõppmädanik** | Must laik vilja tipus | Ühtlane kastmine, kaltsium |

### Kahjurid

| Kahjur | Tunnused | Tõrje |
|--------|----------|-------|
| **Lehetäid** | Keerdunud lehed, kleepuv kaste | Seebivesi, neebiõli |
| **Karilane** | Valged kärbsed lehtede all | Kollased liimpüünised |
| **Kedriklest** | Kollased lehed, peen võrk | Pihusta vett, väävlipreparaat |

### Ennetamine

- **Külvikord:** Ära kasva tomateid samas kohas 3–4 aastat
- **Õhuring:** Eemalda alumised lehed, hõrenda
- **Kastmine:** Ainult mulda, mitte lehti
- **Hügieen:** Eemalda haigeid taimi kohe
- **Multš:** Hoiab ära mullapritsmed lehtedele

## Tolmlemine kasvuhoones

Kasvuhoones pole palju tolmeldajaid – aita kaasa:

1. **Raputamine:** Raputa õitsevaid taimi kergelt keskpäeval
2. **Vibratsioon:** Elektriline hambahari õite juures
3. **Ventilatsioon:** Hea õhuvahetus aitab tolmu levikule
4. **Kimalased:** Võib tuua kimalasepere (suurtes kasvuhoonetes)

## Saagi koristamine

### Küpsuse märgid

| Märk | Kirjeldus |
|------|-----------|
| Värv | Ühtlane sortomane värv |
| Kõvadus | Kergelt järeleandlik |
| Lõhn | Aromaatne tomati lõhn |

### Koristamine

- Korja regulaarselt (iga 2–3 päeva)
- Korja hommikul pärast kaste kuivamist
- Jäta väike varreots külge
- Rohelised viljad – valmivad toas (banaaniga kiirem)

### Hooaja lõpp

- **September lõpp:** Näpista ladvad (viimased viljad küpsevad)
- **Enne öökülmi:** Korista kõik viljad (ka rohelised)
- **Rohelised viljad:** Toas valmima (15–20°C, pimedas)
