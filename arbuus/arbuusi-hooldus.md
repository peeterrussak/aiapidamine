# Arbuusi kasvatamine kasvuhoones

Arbuus (*Citrullus lanatus*) on kõrvitsaline (Cucurbitaceae), mis Eestis vajab **kasvuhoone soojust**. Avamaal kasvuaeg liiga lühike.

---

## Ülevaade

| Parameeter | Väärtus |
|------------|---------|
| Asukoht | Kasvuhoone (avamaal liiga riskantne!) |
| Ettekasvatamine | Mai algus (4–5 nädalat enne istutamist) |
| Istutamine kasvuhoonesse | Juuni algus |
| Saak | August–september |
| Ruumivajadus | 2–3 m² taime kohta |
| Tolmeldamine | **Käsitsi** (putukad ei jõua kasvuhoonesse) |
| Viljad taime kohta | **Piira 2–3 vilja** (suuremad, magusamad) |

---

## Sordid

| Sort | Vili | Kasvuaeg | Märkused |
|------|------|----------|----------|
| **'Sugar Baby'** ⭐ | Ümmargune, punane liha | 75–80 päeva | Levinuim kasvuhoonesort Eestis, kompaktne |
| **'Golden Midget'** | Kollane koor, punane liha | 70 päeva | Väike vili (1–2 kg), varajane |
| **'Orangeglo'** | Oranž liha | 85 päeva | Erinev, magus |
| **'Mini Love F1'** | Väike, punane | 70 päeva | Kompaktne kasvuviis |

**Soovitus:** 'Sugar Baby' – kõige katsetatum ja usaldusväärsem Eesti kasvuhoonetes.

---

## Ettekasvatamine

### Ajakava

| Etapp | Aeg |
|-------|-----|
| Seemnete külv | **Mai algus** |
| Idanemine | 5–7 päeva (25–30°C) |
| Kasvuaeg sees | 4–5 nädalat |
| Istutamine kasvuhoonesse | **Juuni algus** |

### Külv

| Tegur | Juhis |
|-------|-------|
| Pot | Üks seeme potti (ei talu ümberistutust!) |
| Sügavus | 2–3 cm |
| Mullatemperatuur | 25–30°C (kasuta soojendust kui vaja) |
| Kasvukoht | Valge, soe (aknalaud, kasvulamp) |

**⚠️ Arbuus ei talu juurte häirimist** – ista otse lõplikku potti või kasuta turbapotte, mille saab mullaga koos maha panna.

---

## Istutamine kasvuhoonesse

| Tegur | Juhis |
|-------|-------|
| Aeg | Juuni algus (muld min 18°C, öökülmad möödas) |
| Temperatuur kasvuhoones | Min 15°C öösel |
| Vahe | 80–100 cm (horisontaalne) |
| Asukoht | Lõunapoolne kasvuhooneosa, mitte varjus |
| Ruumivajadus | 2–3 m² taime kohta |

### Kasvuhoone paigutus

```
KASVUHOONE (3 × 6 m näide):
┌─────────────────────────────────────────┐
│  Tomatid (vertikaalselt, põhjas)        │
│  ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑     │
│                                         │
│  Kurgid (vertikaalselt, küljel)         │
│  ↑↑↑↑↑↑↑                               │
│                                         │
│  ●           ●                          │
│  Arbuus    Arbuus   ← horisontaalselt   │
│  (2–3 m² ala)                           │
│                                         │
└─────────────────────────────────────────┘
```

Arbuus kasvab horisontaalselt – vajab põranda pinda, mitte vertikaalset tuge (v.a. väikeste sortide puhul, kus võib kasutada kandevõrku).

---

## Hooldus

### Tolmeldamine – kriitiliselt tähtis!

Kasvuhoones **ei ole tavaliselt piisavalt putukaid** tolmeldamiseks. Tolmeldada käsitsi!

**Kuidas:**
1. Leia **isasõis** (sirge varrekesega, ei paistka munasarj)
2. Leia **emasõis** (tüpakasõis – väike arbuus munasarjana all)
3. Võta pintsel või vatitups
4. Kanna isasõie tolm üle emasõiele
5. Korda **iga päev** kuni 3–4 päeva järjest (kuni õis kinnib)

```
ISASÕIS vs EMASÕIS:
                              väike arbuus
  Isasõis:          Emasõis:  munasarjana
     🌸                 🌸
      │                 │
      │ (sirge)         ╰─○ ← munasarj all
      │
```

**⚠️ Ilma tolmeldamiseta ei moodusta arbuus vilja!**

### Viljade piiramine

- **Jäta ainult 2–3 vilja** taime kohta
- Eemalda ülejäänud viljakesed varakult (alla 5 cm)
- Vähem vilju = suuremad, magusamad arbuusid

```
VILJADE PIIRAMINE:
  Taim                    Jäta alles
  ╭──────────────────────╮
  │  ○ ○ ○ ○ ○ ○ ○ ○ ○  │  → Jäta 2–3 ○
  │  (palju väikseid)    │     Eemalda ülejäänud
  ╰──────────────────────╯
```

### Kasvatusviis

**Horisontaalne (tavaline):**
- Lase kasvada maapinnal
- Pane kasvavad viljad **laudkivi, põhualuse** või võrkkotti peale (kaitseb niiskuskahjustuse ja mädaniku eest)

**Vertikaalne (väikestele sortidele):**
- Lase ronida võrgu najal
- Kasuta **kandevõrku** iga viljakülje all (nagu kott)
- Sobib 'Sugar Baby' suuruses viljadele

## Kastmine

| Periood | Sagedus | Märkused |
|---------|---------|----------|
| Istutamise järel | Regulaarselt | Kuni juured kinni |
| Kasvuperiood | 2–3× nädalas | Sügav kastmine |
| Õitsemise ajal | Regulaarselt | Oluline viljakandmiseks |
| Vilja kasv | 2× nädalas | Ühtlane niiskus |
| **Viimased 2 nädalat enne saaki** | **Vähenda oluliselt** | **⭐ Kriitilisim samm – kuivus = magusam arbuus!** |

**⚠️ Ülekastmine lõpuperioodil teeb arbuusi vesiseks ja magedaks!**

## Temperatuur ja kliima

| Parameeter | Optimaalne | Minimaalne |
|------------|-----------|-----------|
| Päeval | 25–32°C | 18°C |
| Öösel | 18–22°C | 15°C |
| Muld | min 18°C | 15°C |

- **Ventilatsioon** oluline – arbuus talub kuumust, aga ülekuumenemine (>38°C) kahjustab
- Avage kasvuhoone aknad ja uksed kuumadel päevadel

| Aeg | Väetis | Märkused |
|-----|--------|----------|
| Istutamisel | Kompost auku | Hea algus |

| Viljakandmise ajal | Kaalium + fosfor | Mitte liiga palju lämmastikku (lehed, mitte viljad) |
| Viimased 2–3 nädalat | Ei vajalik | |

## Kahjurid ja haigused

| Probleem | Tunnused | Ennetamine/tõrje |
|----------|----------|------------------|
| **Lehetäid** | Keerdunud lehed, kleepuvad kolooniad | Seebivesi, neebiõli |
| **Punaämblik** | Peen võrgustik lehtedel, kollased laigud | Kasvuhoone niistuta, neebiõli; kuiv õhk soodustab |
| **Jahukaste** | Valge jahu-taoline kate lehtedel | Hea ventilatsioon, ära kasta lehtedele |
| **Juuremädanik** | Närbunud taim, pruunid juured | Mitte ülekasta, hea drenaaž |
| Viljakandmise ebaõnnestumine | Viljad ei kasva, langevad | Tolmeldamine käsitsi! |
| Vesine maitse | Mahe, vedel liha | Vähenda kastmist 2 nädalat enne saaki |

## Saagi küpsuse märgid

| Märk | Kirjeldus |
|------|-----------|
| **Kõhuvine** | Koputamisel kõlav, tühi heli (nagu tühi tünn) |
| **Tendrils** | Oksake vilja lähedal (kõige lähem) kuivab ja pruunistub |
| **Alumine plekk** | Maapinna poolne osa muutub kollakaks (rohelt) |
| **Varr** | Kuivab koore lähedal |
| **Aeg** | ~75–80 päeva pärast tolmeldamist |

**⚠️ Liiga vara korjamine = maitsetu arbuus.** Kontrolli mitut märki korraga!

```
KÜPSUSE KONTROLL:
         ↓ tendril kuivab?
  ──────╮
        │  tendrilvõrse
  ──────╯
        🍉  ← kopu: kõlav heli = küps
              alumine pool = kollakad laigud
```

## Aastaringne kalender

| Kuu | Tegevus |
|-----|---------|
| **Mai algus** | **Seemnete külv** (potti, 25–30°C) |
| Mai | Ettekasvatamine (valge, soe koht) |
| **Juuni algus** | **Istutamine kasvuhoonesse** |
| Juuni | Esimeste õite ilmumisel alusta **käsitsi tolmeldamist** |
| Juuni–juuli | Piira viljad (jäta 2–3), kastmine, väetamine |
| Juuli–august | Kastmise vähendamine saagi lähenedes |
| **August–september** | **Saagi koristamine** |
| September | Kasvuhoone puhastamine, kompostile |
