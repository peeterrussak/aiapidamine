# Porgandi kasvatamine

Porgand (*Daucus carota*) on P3 peenra põhikultuur. Kasvab kogu hooaja – aprillist oktoobrini.

---

## Ülevaade

| Parameeter | Väärtus |
|------------|---------|
| Asukoht 2026 | Peenar 3 (koos sibula, porru, hernega) |
| Külviaeg | Aprill keskpaik – mai algus |
| Saak | August–oktoober (varajased august, hilised oktoober) |
| Idanemisaeg | 14–21 päeva (aeglane!) |
| Parim naaber | Sibul, murulauk (peletavad porgandikärbest) |
| Väldi kõrval | Till (pidurdab kasvu) |

---

## Sordid

| Sort | Tüüp | Pikkus | Märkused |
|------|------|--------|----------|
| **'Nantes'** ⭐ | Klassika | 15–20 cm | Silindriline, magus, levinuim |
| **'Jõgeva Nantes'** ⭐ | Kohalik | 15–18 cm | Eesti sort, hea säilivus |
| **'Autumn King'** | Hiline | 20–25 cm | Talveporgand, pikk säilivus |
| **'Paris Market'** | Ümar | 3–5 cm | Raske/kivise mulla jaoks, kiire |
| **'Chantenay'** | Kooniline | 10–15 cm | Raskemale mullale, saagikas |

**Soovitus:** 'Jõgeva Nantes' on kindlaim valik Eesti tingimustes.

---

## Külv

Porgand **ei vaja ettekasvatamist** – külvatakse otse peenrasse.

### Külviaeg

| Aeg | Sobivus | Märkused |
|-----|---------|----------|
| **Aprill keskpaik** | ⭐ Parim | Muld min 7°C |
| Mai algus | Hea | Kindlam idanemine |
| Mai–juuni | Järelkülv | Pikem saagiaeg |

### Külvijuhised

| Tegur | Juhis |
|-------|-------|
| Sügavus | 1–2 cm (väga pinnapealne!) |
| Vahe reas | 3–5 cm (harvendada 5–8 cm peale) |
| Ridade vahe | 20–25 cm |
| Mullatemperatuur | Min 7°C (optimaalne 16–22°C) |

**Idanemise kiirendamine:**
- Leota seemneid 24h enne külvi
- Kata rida kattelooriga (hoiab niisket, kiirendab idanemist)
- Eemalda katteloor **kohe**, kui tõusmed nähtavad

**Hea nipp:** Külva porgandiga koos redist samasse ritta. Redis idaneb kiiresti (3–5 päeva), märgistab aeglaselt idaneva porgandi rea ja korjatakse välja enne, kui porgand ruumi vajab.

```
PORGAND + REDIS KOOS:
┌──────────────────────────────────────────────────────┐
│  P  r  P  r  P  r  P  r  P  r  P  r  P  r  P  r  P  │
│                                                      │
│  P = porgand (aeglane, jääb alles)                   │
│  r = redis (kiire, korjatakse ~25 päeva pärast)       │
└──────────────────────────────────────────────────────┘
```

### Harvendamine

**⚠️ Harvendamine on kohustuslik!** Tihedalt kasvanud porgandid jäävad väikseks.

- Harvendam kui tõusmed 3–5 cm kõrged
- Jäta 5–8 cm vahe taimede vahele
- Harvendatud pisikesed porgandid on söödavad!

## Paigutus peenras P3

```
┌────────────────────────────────────────────────────────────┐
│ Muru-  │  HERNED + OAD   │    PORGAND     │  SIBUL + PORRU │ Muru-│
│ lauk   │  (kevad–suvi)   │  4 rida, 2,5m  │   (2,5–3 m)    │ lauk │
│        │  ← 3 m →        │   ← 2,5 m →    │    ← 3 m →     │      │
└────────────────────────────────────────────────────────────┘

Pärast herneid (juuli): asenda herned salatite/spinatiga!
```

## Hooldus

### Kastmine

| Periood | Sagedus | Märkused |
|---------|---------|----------|
| Idanemine | Regulaarselt niiske | **Kriitilisim periood** – ei tohi kuivada! |
| Kasvuperiood | 1× nädalas | Sügav kastmine (10–15 cm) |
| Suvi | Vastavalt ilmale | Ühtlane niiskus = ühtlane kasv |
| **Ebaühtlane kastmine** | – | ⚠️ Põhjustab porgandite hargnemist ja lõhenemist! |

### Väetamine

| Tegur | Juhis |
|-------|-------|
| Kompost | Pane sügiseti multšiks (mitte kevadel otse mulda) |
| Lämmastik | **Mitte liiga palju** – liigne lämmastik = palju lehti, hargnenud juur |
| Värske sõnnik | **Väldi** – põhjustab hargnemist |
| Kaalium | Kasulik (puutuhk 50 g/m²) – parandab maitset |

### Rohimine

- Esimesed 4–6 nädalat on rohimine oluline (porgand on nõrk konkurent)
- Kui taimed katvamad, umbrohi ei häiri enam

### Multšimine

- Multši porgandiridu heinaga (5 cm kiht)
- Hoiab niiskust, pärsib umbrohtu, ühtlustab mullatemperatuuri

## Kaaslaskultuurid

| Kaaslane | Kasu |
|----------|------|
| **Sibul** ⭐ | Klassikaline paar – sibul peletab porgandikärbest |
| **Murulauk** ⭐ | Sama efekt nagu sibulal, püsiv |
| **Redis** | Märgistab rea, korjatakse enne (ei võta ruumi) |
| **Herned** | Annavad lämmastikku mulda |
| **Salat** | Täidab tühja ruumi peenras |

| Halb naaber | Põhjus |
|-------------|--------|
| **Till** | Pidurdab porgandi kasvu |
| **Apteegitill (fennel)** | Kahjulik kaaslane |

## Kahjurid ja haigused

| Probleem | Tunnused | Ennetamine/tõrje |
|----------|----------|------------------|
| **Porgandikärbes** | Roostes pruunikad käigud juurikas, lehed kollaseks | Sibul ja murulauk naabrina, katteloor maist juunini, külvikord |
| **Lehetäid** | Keerdunud lehed | Seebivesi, lepatriinud |
| **Alternarioos** | Mustad laigud lehtedel | Külvikord, hea õhuring |
| Hargnenud juur | Kaks või rohkem harunemist | Väldi värsket sõnnikut, ühtlane kastmine |
| Lõhenenud juur | Pikisuunaline lõhenemine | Ühtlane kastmine, ära kastmine pärast põuda liiga äkki |

**Porgandikärbes** on peamine oht. Kärbse lennuperiood on mai–juuni. Katteloor selle perioodi jooksul kaitseb hästi.

## Koristamine

### Millal korjata

| Märk | Kirjeldus |
|------|-----------|
| Kael | Porgandi ülaosa nähtav maapinnast (5–8 mm) |
| Aeg | August (varajased), september–oktoober (hilised) |
| Maitse | Esimeste öökülmade järel paraneb maitse (suhkrusisaldus kasvab) |

**Varajane korjamine (august):** Korja osa porgandeid noorelt – maitsvamad värskelt.

**Talveporgand:** Sordid nagu 'Autumn King' jäta mulda – kaitse paksult multšiga ja korja vastavalt vajadusele kuni novembrini.

### Koristamine

1. Kasta päev enne (muld pehmem)
2. Tõmba juurika ülaosast, mitte lehtedest
3. Lõika lehed 2 cm pealt (hoiab niiskust)
4. ÄRA pese enne säilitamist

---

## Säilitamine

| Viis | Temperatuur | Kestus | Märkused |
|------|-------------|--------|----------|
| **Mullas** (parim!) | – | Oktoober–november | Multši paksult, korja vastavalt vajadusele |
| **Liivas keldris** | 0–4°C | 4–6 kuud | Kihiti liivas või turbas |
| Külmkapp | 0–4°C | 3–4 nädalat | Kotti, eemal puuviljadest |
| Külmutamine | -18°C | 8–10 kuud | Blanšeeri enne! |
