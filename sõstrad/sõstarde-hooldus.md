## Sõstratüübid

| Tüüp | Ladina nimi | Maitse | Kasutus | Märkused |
|------|-------------|--------|---------|----------|
| **Must sõstar** | Ribes nigrum | Intensiivne, aromaatne | Moos, mahl, liköör, värskelt | Kõige aromaatsem, C-vitamiini rikas |
| **Punane sõstar** | Ribes rubrum | Hapu, värskendav | Tarretis, kaunistus, mahl | Ilus, pikk kobaras |
| **Valge sõstar** | Ribes rubrum (valge vorm) | Mahe, magusam | Värskelt, kaunistus, moos | Punase sõstra valge vorm |

## Erinevused hoolduses

| Omadus | Must sõstar | Punane/valge sõstar |
|--------|-------------|---------------------|
| **Viljumine** | Eelmise aasta okstel (1–2 a) | Vanemal puidul (2–3 a) |
| **Lõikus** | Agressiivsem, uuenda 1/3 igal aastal | Õrnem, säilita vanemaid oksi |
| **Kasv** | Tugevam, lopsakam | Kompaktsem |
| **Valmimisaeg** | Juuli algus–keskpaik | Juuli keskpaik–august |
| **Marjakobara pikkus** | Lühem | Pikem, dekoratiivsem |

## Kasvukoha valik

### Ideaalne kasvukoht

| Tegur | Nõue | Märkused |
|-------|------|----------|
| **Valgus** | Päike kuni poolvarjus | Must sõstar talub rohkem varju |
| **Muld** | Viljakas, niiske, hästi dreenitud | pH 6,0–6,5 |
| **Niiskus** | Ühtlane, ei talu põuda | Eriti oluline marjade valmimise ajal |
| **Tuul** | Varjatud koht parem | Tuul kuivatab ja kahjustab õisi |

**⚠️ Väldi:**
- Liiga kuiva ja liivast mulda
- Madalaid külmade kogumise kohti (hiliskülmakahjustused!)
- Tuulist avatud kohta

## Istutamine

### Istutusaeg

| Aeg | Sobivus | Märkused |
|-----|---------|----------|
| **Sügis (okt–nov)** | ⭐ Parim | Juurdub enne talve, kevadel kasvab kohe |
| **Varakevad (märts–apr)** | Hea | Enne pungade puhkemist |

### Istutuskaugused

| Tüüp | Vahekaugus reas | Ridade vahe |
|------|-----------------|-------------|
| Must sõstar | 1,2–1,5 m | 2,0–2,5 m |
| Punane/valge sõstar | 1,0–1,2 m | 1,8–2,0 m |

### Istutamise sammud

1. **Kaeva auk** – 2× juurepalli laius, 40–50 cm sügav
2. **Lisa komposti** – sega kaevatud mullaga
3. **Istuta 5–10 cm sügavamale** kui taimlas kasvas
   - Soodustab uute võrsete teket juurepõhjast
4. **Kasta põhjalikult** – 10–15 liitrit
5. **Multši** – 10 cm kiht (mitte vastu tüve)
6. **Lõika tagasi** – jäta 3–4 silma (pungas) oksa kohta

## Lõikuse põhimõtted

### Must sõstar – lõikus

**Põhimõte:** Must sõstar kannab parimaid marju **1–2 aasta vanustel okstel**. Vanemad oksad annavad vähem ja väiksemaid marju.

**Iga-aastane lõikus (varakevad või pärast saaki):**

1. **Eemalda:**
   - Üle 3 aasta vanad oksad (tumedad, jämedad, kooritud)
   - Maapinnal lamavad oksad
   - Haigeid, kahjustatud, ristuvaid oksi
   - Nõrgad, peenikesed võrsed

2. **Jäta:**
   - 8–12 tugevat oksa põõsa kohta
   - Eri vanusega: ~4 uut + ~4 kaheaastast + ~4 kolmeaastast
   - Põõsa keskosa avatuks (õhuring!)

```
MUSTA SÕSTRA LÕIKUS:

Enne:                     Pärast:
    |  |  |  |  |             |  |  |
    |  ╲  |  ╱  |             |  |  |
    ╲  ╲  |  ╱  ╱             |  |  |
─────┴──┴──┴──┴──┴─────    ───┴──┴──┴───
  vana  vana  vana            uued +
  + uued ristuvad            2-aastased

→ Eemalda 1/3 vanematest okstest igal aastal
→ Jäta keskosa avatuks
```

### Punane ja valge sõstar – lõikus

**Põhimõte:** Punane/valge sõstar kannab parimaid marju **2–3 aasta vanustel okstel**. Oksad on produktiivsed kauem kui mustal sõstral.

**Iga-aastane lõikus (varakevad):**

1. **Eemalda:**
   - Üle 4 aasta vanad oksad
   - Maapinnal lamavad oksad
   - Haigeid ja kahjustatud oksi
   - Liiga tihedalt kasvavaid võrseid

2. **Jäta:**
   - 8–10 peaoksa
   - Lühenda külgharusid 2–3 pungani (soodustab marjakobara arengut)
   - Põõsa kuju avatum kui mustal sõstral

```
PUNASE/VALGE SÕSTRA LÕIKUS:

      ╱ ╲                   ╱ ╲
     ╱   ╲                 ╱   ╲
    ╱     ╲               ╱     ╲
───┴───────┴───        ──┴───────┴──
    pokaal-kuju           avatud
                          keskosaga

→ Säilita vanemaid produktiivseid oksi
→ Lühenda külgharud 2–3 pungani
→ Eemalda üle 4 a vanad oksad
```

## Aastaringne hoolduskalender

### Varakevad (märts–aprill)

**Lõikus (enne pungade paisumist!):**
- [ ] Eemalda vanad, haigeid ja kahjustatud oksad
- [ ] Must sõstar: eemalda 1/3 vanematest okstest
- [ ] Punane/valge: lühenda külgharusid
- [ ] Hõrenda põõsa keskosa

**Väetamine:**
- [ ] Lisa kompost (5–10 L põõsa kohta)
- [ ] Või kõdunud sõnnik (3–5 L põõsa kohta)
- [ ] Universaalne marjaväetis (vastavalt pakendile)

**Muu:**
- [ ] Kontrolli talvekahjustusi
- [ ] Paranda tugesid (kui kasutad)

### Kevad (mai)

- [ ] Kontrolli kahjureid (lehetäid, lehevaablased)
- [ ] Multši (hein, hakitud koor) – 5–10 cm
- [ ] Alusta kastmist, kui kuiv kevad
- [ ] Jälgi hiliskülma – kata kattelooriga, kui öökülm ähvardab õitsemise ajal!

**⚠️ Hiliskülm:** Sõstrad õitsevad vara (mai algus) ja õied on külmatundlikud! –2°C võib hävitada saagi.

### Õitsemine (mai)

- [ ] Jälgi tolmeldajaid
- [ ] Ära kasta õitele
- [ ] Kata, kui öökülm ennustatakse

### Suvi – saagiaeg (juuli–august)

**Valmimise järjekord:**
```
Must sõstar → Punane sõstar → Valge sõstar
(juuli algus)   (juuli keskp.)   (juuli lõpp)
```

**Korjamine:**
- [ ] Korja kobaratena (mitte marju eraldi)
- [ ] Must sõstar: kui marjad on pehmed ja tumedatmustad
- [ ] Punane/valge: kui kogu kobar on ühtlaselt värviline
- [ ] Korja kuival ilmal, hommikul

### Pärast saaki (august)

- [ ] Võib teha kergema lõikuse (valikuline)
- [ ] Väeta kaaliumiga (puutuhk, ~50 g põõsa kohta)
- [ ] Jätka kastmist, kui kuiv
- [ ] Jälgi kahjureid ja haigusi

### Sügis (september–november)

- [ ] Kogu langenud lehed kokku (haiguste ennetamine)
- [ ] Multši tüve ümbrus talveks
- [ ] Kui kuiv sügis – kasta enne külmi läbi
- [ ] Uute põõsaste istutamine (oktoober–november)

### Talv

- [ ] Sõstrad on väga külmakindlad (kuni –40°C)
- [ ] Erilist kaitset ei vaja
- [ ] Lumekate on kasulik

## Kahjurid ja haigused

### Kahjurid

| Kahjur | Tunnused | Tõrje |
|--------|----------|-------|
| **Sõstra-klaastiib** | Oksad närbuvad, sees käigud | Lõika nakatunud oksad 10 cm terve koeni, põleta |
| **Sõstra-lehekirp** | Punakad/kollased kühm-muhud lehtedel | Talvine rapiõli, kevadel püretriinisprei |
| **Lehetäid** | Keerdunud lehed, kleepuv pind | Seebivesi, neebiõli, tugev veejuga |
| **Sõstra-pungalest** | Ümarad, pundunud pungad | Eemalda nakatunud pungad, raske korral põõsas hävitada |
| **Karusmarja-vahukoi** | Lehed keeratud kokku, tõugud sees | Lõika nakatunud otsad |
| **Linnud** | Söödud marjad | Kata võrguga |

### Haigused

| Haigus | Tunnused | Ennetamine/tõrje |
|--------|----------|------------------|
| **Jahukaste** (eriti mustal) | Valge jahune kate lehtedel/võrsetel | Hea õhuring, resistentsed sordid, väävlipreparaat |
| **Lehelaik-haigused** | Pruunid/mustad laigud | Kogu lehed sügisel, vaskpreparaat varakevadel |
| **Hallhallitus** | Hallitanud marjad | Hea õhuring, ära kasta marju |
| **Rooste** | Oranžid täpid lehtedel | Eemalda nakatunud lehed |
| **Sõstra-reversiooniviirus** | Moonutatud lehed, ei vilja | Hävita nakatunud põõsas, osta viirusvaba materjal |

### Ennetamine

1. **Hea õhuring** – hõrenda põõsaid regulaarselt
2. **Resistentsed sordid** – eriti jahukaste vastu
3. **Hügieen** – kogu lehed sügisel, eemalda haigeid oksi
4. **Külvikord** – ära istuta sõstraid vana sõstrapõõsa kohale
5. **Terve istutusmaterjal** – osta sertifitseeritud taimlast

## Marjade korjamine

### Õige aeg

| Tüüp | Valmimise märgid | Korjamisaeg |
|------|------------------|-------------|
| Must sõstar | Marjad pehmed, läikivmustad, kergelt maas | Juuli algus–keskpaik |
| Punane sõstar | Kogu kobar ühtlaselt punane, läikiv | Juuli keskpaik–lõpp |
| Valge sõstar | Kobar läbipaistev-kollakas, pehme | Juuli lõpp–august |

### Korjamise nipid

**Must sõstar:**
- Marjad võivad valmida ebaühtlaselt
- Korja mitme käiguga (1–2 nädala jooksul)
- Korja kobarana või marju eraldi

**Punane/valge sõstar:**
- Korja KOGU KOBAR korraga (sõrmedega kobara aluse katkestamine)
- Ära tõmba marju ükshaaval – vigastavad
- Kobarad säilivad paremini kui lahtised marjad

### Korjamissagedus

- Korja iga **3–5 päeva**
- Korja **kuival ilmal**, hommikul pärast kaste kuivamist
- Kasuta madalaid anumaid (marjad ei muljuks)

## Marjade säilitamine

| Viis | Säilivus | Juhend |
|------|----------|--------|
| **Värskelt** (külmikus) | 3–5 päeva | Ära pese, hoia kobarana |
| **Külmutamine** | 12 kuud | Pese, kuivata, külmuta laiali, paki kottidesse |
| **Moos** | 12+ kuud | Must sõstar 1:1 suhkruga; punane/valge 1:0,8 |
| **Mahl** | 12 kuud | Pressi, pastöriseeri 85°C |
| **Tarretis** (punane!) | 12+ kuud | Punane sõstar on looduslikult pektiinirikas |
| **Kuivatamine** | 12 kuud | Kuivatis 55–60°C |
| **Siirup** | 12 kuud | Keeda mahlaga, lisa suhkur |

### Kasutusideed

**Must sõstar:**
- Moos, mahl, siirup
- Liköör (Crème de Cassis)
- Smuutid
- Küpsetised

**Punane sõstar:**
- Tarretis (ideaalne looduslik pektiin!)
- Kaunistus tortidele
- Kastmed lihatoitudele
- Moos, mahl

**Valge sõstar:**
- Värskelt (kõige magusam!)
- Kaunistus
- Moos
- Vahuveinid, kokteilid

## Paljundamine

### Puitunud pistokstest (lihtsaim!)

**Aeg:** Oktoober–november (või varakevad)

**Sammud:**
1. Lõika terve 20–25 cm oks (sel aasta kasv)
2. Lõika ülemine ots kaldus (vee äravool), alumine sirge
3. Eemalda alumised pungad, jäta ülemised 3–4
4. Pista 2/3 ulatuses mulda (kaldega mõne kraadi võrra, pungad peale)
5. Tihenda muld ümber
6. Juurdumine kevadeks

**Edu määr:** 70–90%

### Kätega maha painutamine

1. Painuta alumine oks maha
2. Kinnita traadiklambriga, kata mullaga (keskosa)
3. Juurdumine ~1 aasta
4. Lõika emapõõsast lahti, istuta ümber

## Probleemide lahendamine

| Probleem | Võimalik põhjus | Lahendus |
|----------|-----------------|----------|
| Vähe marju | Hiliskülm hävitas õied | Kata õitsemise ajal |
| Vähe marju | Vana põõsas | Uuenda lõikusega või istuta uus |
| Vähe marju | Liiga varjuline koht | Istuta päiksele |
| Väikesed marjad | Põud | Kasta regulaarselt |
| Väikesed marjad | Liiga palju oksi | Hõrenda |
| Kollased lehed | Kloroos (rauapuudus) | Lisa kompost, korrigeeri pH |
| Punased muhud lehtedel | Sõstra-lehekirp | Rapiõli talvel, püretriinisprei |
| Pundunud pungad | Pungalest | Eemalda nakatunud pungad |
| Jahukaste | Seenhaigus | Resistentsed sordid, õhuring |
