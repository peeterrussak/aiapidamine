# Kurkide kasvatamine

## Ülevaade

Kurk (Cucumis sativus) on kõrvitsaliste (Cucurbitaceae) sugukonda kuuluv soojuslembeline kultuur. Eestis kasvatatakse kurke nii kasvuhoones (pikad salatikurgid) kui avamaal (lühemad, hapendamiseks sobivad).

**Kasvuhoone vs avamaa:**

| Omadus | Kasvuhoone | Avamaa |
|--------|------------|--------|
| Sordid | Pikad salatikurgid (25–45 cm) | Lühikesed (8–15 cm), hapendamiseks |
| Saak | Suurem, pikem hooaeg | Väiksem, lühem hooaeg |
| Hooldus | Nõuab rohkem tähelepanu | Lihtsam |
| Riskid | Kahjurid kasvuhoones | Ilmastik, jahukaste |
| Istutusaeg | Mai keskpaik | Juuni algus |

**2025 kogemus:**
- 4–5 kurki taime kohta

**Eripära:**
- Hiina traditsiooniline sort
- Mõruvaba (ei lähe kibedaks)
- Õhuke, söödav koor
- Sobib salatitesse ja värskelt söömiseks

## Kasvatamine

### Ettekasvatamine

Kurgid on tundlikud ümberistutamisele – külva otse suuremasse potti!

| Tegevus | Aeg | Juhis |
|---------|-----|-------|
| Külv | 3–4 nädalat enne õue istutamist | |
| | Kasvuhoone: aprill lõpp – mai algus | |
| | Avamaa: mai algus | |
| Temperatuur | 22–28°C idanemisel | Kasuta soojendusmatsi |
| Idanemine | 5–7 päeva | Niiske, mitte märg |
| Potid | 8–10 cm läbimõõduga | Külva 2 seemet, jäta tugevam |

**⚠️ Tähtis:** Kurgid EI TALU juurestiku häirimist! Istuta potiga koos.

### Istutamine

**Kasvuhoonesse:**

| Tegevus | Juhis |
|---------|-------|
| Aeg | Mai keskpaik–lõpp (muld >15°C) |
| Taimede vahe | 40–50 cm |
| Ridade vahe | 100–120 cm |
| Taimi | ~4–6 taime 3 m² kohta |

**Avamaale:**

| Tegevus | Juhis |
|---------|-------|
| Aeg | Juuni algus (öökülmad möödas, muld min 15°C) |
| Taimede vahe | 60–80 cm |
| Ridade vahe | 100–120 cm |
| Koht | Päikeseline, tuulevarju |

## Hooldus

### Kasvuhoone kurkide hooldus

**Tugi ja vormilõikus:**

```
VERTIKAALNE KASVATAMINE
═══════════════════════════════════════  Lakk (2–2,5 m)
│         │         │         │
│ nöör    │  nöör   │  nöör   │         Nöörid lakke
│         │         │         │
🥒        🥒        🥒        🥒         Kurgi taimed
──────────────────────────────────────── Maapind

    50cm      50cm      50cm            Vahed
```

- **Tüvi:** Eemalda esimesed 4–5 külgvõrset ja viljad (tugevdab taime)
- **Ladvamine:** Näpista ladv kui jõuab lae alla
- **Külgvõrsed:** Näpista pärast 2. lehte

**Avamaa kurkide hooldus:**
- Võib kasvatada maas (mulši viljade alla)
- Või madala võrgu najal (puhtamad viljad)
- Võrseid ei pea nii palju pügama kui kasvuhoones

### Kastmine

| Koht | Sagedus | Märkused |
|------|---------|----------|
| Kasvuhoone | Iga päev või ülepäeviti | Rohkelt, aga mulda, mitte lehtedele |
| Avamaa | 2–3× nädalas | Sõltub ilmast, kuival perioodil sagedamini |

**⚠️ Allakasmise märgid:**
- Lehed longus
- Kurgid kibed!
- Viljad kõverduvad

**Kastmise reeglid:**
- Kasta MULDA, mitte lehti (seenhaiguste vältimiseks)
- Kasta hommikul (lehed jõuavad kuivada)
- Kasuta toasooja vett (mitte külma kraanivett)

### Väetamine

| Aeg | Väetis | Kogus |
|-----|--------|-------|
| Istutamisel | Kompost mulda | 5–10 cm kiht |
| Õitsemisest alates | Nõgesevesi | 1:10 lahjendus, iga 10–14 päeva |
| Viljakandmise ajal | Nõgesevesi või köögiviljaväetis | Iga 2 nädalat |

## Korjamine

### Millal korjata

**Kasvuhoone (pikad sordid):**
- Korja 25–35 cm pikkuselt (parim maitse ja tekstuur)
- 'Suyu Long' parim 25–30 cm juures

**Avamaa (lühikesed sordid):**
- Hapendamiseks: 6–10 cm (kornišonid)
- Värskelt: 10–15 cm
- Ära lase üle kasvada (muutub seemneliseks)

### Kuidas korjata

- Korja regulaarselt, **iga 2–3 päeva**
- Kasuta teravat nuga või kääre (ära rebi!)
- Mida rohkem korjad, seda rohkem taim toodab
- Hommikul korjatud kurgid on krõbedamad

### Saagi ootus

| Koht | Saak taime kohta | Hooaja pikkus |
|------|------------------|---------------|
| Kasvuhoone | 15–25 kurki | Juuli – september |
| Avamaa | 10–15 kurki | Juuli lõpp – september |

## Kahjurid ja haigused

### Kahjurid

| Kahjur | Tunnused | Tõrje |
|--------|----------|-------|
| **Lehetäid** | Keerduned lehed, kleepuv kaste | Nõgesevesi, seebi-vesilahus, lepatriinud |
| **Punane kedriklest** | Tillukesed täpid, õhukesed võrgud | Tõsta niiskust, pihusta vett lehtedele |
| **Kasvuhoone karilane** | Valged kärbsed lehtede all | Kollased liimpüünised |
| **Limukad** | Näritavad lehed ja viljad | Õllelõksud, käsitsi korjamine |

### Haigused

| Haigus | Tunnused | Ennetamine/tõrje |
|--------|----------|------------------|
| **Jahukaste** | Valge pulbertaoline kate lehtedel | Hea õhuring, ära kasta lehti, eemalda haiged lehed |
| **Kurgi-ebajahukaste** | Kollased laigud (kandiline muster) | Tuuluta, vasepreparaat |
| **Hallitushaigused** | Hall kate, mädanemine | Tuuluta, väldi üleniiskust |
| **Viirushaigused** | Moonutatud, mosaiiklehed | Eemalda lehetäid, haigeid taimi |

### Ennetamine

1. **Külvikord** – kurgid samasse kohta alles 3–4 aasta pärast
2. **Hea õhuring** – eriti kasvuhoones!
3. **Kasta mulda** – mitte lehtedele
4. **Eemalda haigeid lehti** – kohe, ära komposti
5. **Kaaslaskultuurid** – till meelitab tolmeldajaid, peletab kahjureid

## Säilitamine

### Värskelt

- **Külmkapis:** 1–2 nädalat (köögiviljade sahtlis, mitte liiga külm!)
- **Toatemperatuuril:** 2–3 päeva
- **⚠️ Ära hoia koos õuntega!** (etüleen kiirendab kollaseks minekut)

### Hapendamine

Eesti klassika! Hapukurgid säilivad keldris kuni kevadeni.

**Lihtne retsept (1 L purk):**
1. Pese väikesed kurgid (6–10 cm)
2. Aseta purki: küüslauguküüs, tilliõisikud, musta sõstra lehed, mädarõikaleht
3. Lisa kurgid tihedalt
4. Vala peale leige soolalahus (1 spl soola / 1 L vee kohta)
5. Lase toatemperatuuril kääritada 2–3 päeva
6. Vii keldrisse

### Marineerimine

Äädikaga kurgid säilivad kauem ja on hapukurgidest terava maitsega.
