# Külvikord ja rotatsioon

Kultuuride vaheldumise põhimõtted ja planeerimine.

---

## Miks külvikord?

### Probleemid ilma külvikorrala

| Probleem | Selgitus |
|----------|----------|
| **Haigused kuhjuvad** | Mullas talvituvad haigustekitajad nakatavad sama perekonna taimi |
| **Kahjurid paljunevad** | Spetsialiseerunud kahjurid leiavad toitu igal aastal |
| **Toitained ammenduvad** | Sama kultuur võtab samu toitaineid |
| **Saak väheneb** | Taimed nõrgenevad, haigused levivad |

### Külvikorra kasud

| Kasu | Selgitus |
|------|----------|
| **Haiguste katkestamine** | Haigustekitajad surevad, kui peremeestaimet pole |
| **Kahjurite vähenemine** | Elutsükkel katkeb |
| **Toitainete tasakaal** | Erinevad kultuurid kasutavad erinevaid toitaineid |
| **Mulla struktuur** | Erinevad juuretüübid parandavad mulda |
| **Lämmastiku sidumine** | Liblikõielised rikastavad mulda |

## Kultuurirühmad

### Peamised rühmad

| Rühm | Kultuurid | Eripära |
|------|-----------|---------|
| **Maavitsalised** (Solanaceae) | Tomat, kartul, paprika, baklažaan | Kõrge toitainevajadus, sarnased haigused |
| **Kõrvitsalised** (Cucurbitaceae) | Kurk, kõrvits, suvikõrvits, arbuus, melon | Kõrge toitainevajadus, jahukaste |
| **Kapsalised** (Brassicaceae) | Kapsas, lillkapsas, brokoli, redis, naeris, rüps | Kapsanuuter (!), kõrge N-vajadus |
| **Liblikõielised** (Fabaceae) | Herned, oad | Seovad lämmastikku, parandavad mulda |
| **Sarikõielised** (Apiaceae) | Porgand, pastinaak, seller, till, petersell | Porgandikärbes |
| **Laugulised** (Allium) | Sibul, küüslauk, porru, murulauk | Sibulakärbes |
| **Salati- ja spinatitaimed** | Salat, spinat, mangold | Kiire kasv, madal toitainevajadus |

### Rühmadevahelised seosed

```
PARIM JÄRGNEVUS:

Liblikõielised → Maavitsalised → Kõrvitsalised → Kapsalised
(herned, oad)    (tomat, kartul)   (kurk, kõrvits)  (kapsas)
     ↓                 ↓                 ↓              ↓
  Seovad N        Kasutavad N       Kasutavad N    Kasutavad N
                  (palju!)          (palju!)       (palju!)

Pärast liblikõielisi on mullas rohkem lämmastikku!
```

## Rotatsiooni põhimõtted

### Põhireegel: ära kasva sama rühma taimi samal kohal

| Rühm | Minimaalne paus | Põhjus |
|------|-----------------|--------|
| Maavitsalised | 3–4 aastat | Lehemädanik, fusarioos |
| Kõrvitsalised | 2–3 aastat | Jahukaste, fusarioos |
| Kapsalised | 4–7 aastat | Kapsanuuter (väga püsiv!) |
| Liblikõielised | 2–3 aastat | Juuremädanik |
| Sibulalised | 3–4 aastat | Sibulakärbes, hallitus |
| Sarikõielised | 2–3 aastat | Porgandikärbes |

### 4-aastane rotatsioon (klassikaline)

```
Aasta 1: Liblikõielised (herned, oad) – seo lämmastikku
    ↓
Aasta 2: Maavitsalised (tomat, kartul, paprika) – kasuta N
    ↓
Aasta 3: Kapsalised (kapsas, lillkapsas) – kasuta N
    ↓
Aasta 4: Juurviljad + sibulad (porgand, peet, sibul) – vähe N
    ↓
(Aasta 5: jälle liblikõielised...)
```

### 3-aastane rotatsioon (lihtsustatud)

```
Aasta 1: Lehtkultuurid + liblikõielised (salat, herned, oad, kapsas)
    ↓
Aasta 2: Viljataimed (tomat, kurk, paprika, kõrvits)
    ↓
Aasta 3: Juurviljad + sibulad (porgand, peet, sibul, küüslauk)
    ↓
(Aasta 4: jälle lehtkultuurid...)
```

## Sinu aia rotatsioon (4 peenart)

### Praegune plaan 2025–2028

| Aasta | P1 | P2 | P3 | P4 |
|-------|----|----|----|----|
| **2025** | Maasikad | Herned + oad | Kurgid + kõrvits | Kartul |
| **2026** | Maasikad | Maavitsalised | Liblikõielised + juurviljad | Kõrvitsalised |
| **2027** | Maasikad | Salat + porgand | Maavitsalised | Liblikõielised |
| **2028** | (uuenda maasikad?) | Kõrvitsalised | Kapsalised | Maavitsalised |

### Seletus

**P1 – Maasikad:**
- Maasikad jäävad samale kohale 4–5 aastat
- Pärast uuendamist: paus 4–5 aastat enne uusi maasikaid samale kohale
- Küüslauk vahel aitab haiguste vastu

**P2 rotatsiooni loogika:**
- 2025: Herned seovad lämmastikku
- 2026: Tomatid kasutavad seda lämmastikku
- 2027: Salatid ja porgand vajavad vähem toitaineid
- 2028: Kõrvitsalised saavad eelmise aasta juurviljadest taastunud mulda

**P3 rotatsiooni loogika:**
- 2025: Kurk kasutab toitaineid
- 2026: Liblikõielised taastavad lämmastiku
- 2027: Tomatid kasutavad seda
- 2028: Kapsas vajab palju lämmastikku

**P4 rotatsiooni loogika:**
- 2025: Kartul – tugev toitja
- 2026: Kõrvits – teine tugev toitja
- 2027: Herned – taastavad mulda
- 2028: Tomatid – kasutavad liblikõieliste jäetud N

## Erijuhud

### Maasikad

Maasikad on erand – jäävad samale kohale 4–5 aastat:

| Aasta | Saak | Tegevus |
|-------|------|---------|
| 1 | Väike | Taimed juurduvad |
| 2 | Keskmine | Esimene täissaak |
| 3 | **Suurim** | Parim aasta |
| 4 | Hea | Veel hea, aga langeb |
| 5 | Langeb | Uuenda istandik |

**Pärast maasikaida:**
- Paus 4–5 aastat samale kohale
- Istuta vahele midagi muud (kapsalised, liblikõielised)

### Kartul

Kartul on kõige probleemsem:
- Lehemädanik jääb mulda
- Traatuss armastab kartuleid
- Vajab vähemalt **4 aasta pausi**

**Hea järgnevus:**
```
Kartul → Liblikõielised → Kapsalised → Juurviljad → (Kartul)
```

### Küüslauk

Küüslauk on hea "puhastaja":
- Antibakteriaalsed omadused
- Peletab mõnda kahjurit
- Sobib maasikate vahele

## Planeerimise näpunäited

### Rotatsiooni planeerimine

1. **Joonista kaart** – märgi peenrad ja kultuurid
2. **Pea päevikut** – kirjuta üles, mis kus kasvas
3. **Planeeri 3–4 aastat ette** – väldi vigu
4. **Arvesta erandeid** – maasikad, kartul

### Väikese aia lahendused

Kui rotatsiooni ruum on piiratud:

| Lahendus | Selgitus |
|----------|----------|
| **Pikemad pausid** | Kui ei saa roteerida, oota kauem |
| **Resistentsed sordid** | Vähendavad haigusriski |
| **Mulla vahetamine** | Äärmuslik, aga töötab |
| **Konteinerkasvatus** | Tomat jt potis – värske muld igal aastal |
| **Haljasväetised** | Parandavad mulda vahepeal |

### Levinud vead

| Viga | Tulemus |
|------|---------|
| Kartul ja tomat järjest | Lehemädanik levib |
| Kapsas igal aastal samas kohas | Kapsanuuter! |
| Maasikad liiga kaua samas kohas | Saak langeb, haigused |
| Sibulad pärast sibulaid | Sibulakärbes paljuneb |
