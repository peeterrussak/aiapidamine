## Vaarikate tüübid

### Suvikandvad (floricane)

Sinu sordid ('Glen Ample', 'Glen Dee', must pampel) on **suvikandvad**:
- Viljuvad **eelmise aasta okste** peal
- Lõikus pärast saaki – viljunud oksad maha
- Uued võrsed jäetakse järgmiseks aastaks

### Sügiskandvad (primocane)

Soovitatud: 'Polka', 'Poranna Rosa', 'Delniwa'

- Viljuvad **sama aasta** okste peal
- Saak: august–oktoober (pikendab hooaega!)
- Lõikus: kõik oksad sügisel/kevadel maha
- **Eelis:** Vähem haigusi, lihtsam hooldus

---

## Glen Ample

**Iseloomustus:**
- Üks populaarsemaid sorte Euroopas
- Suured, kindlad marjad
- **Ogadeta varred** – mugav korjata!
- Tugev kasv, vajab tuge

**Saagikus:** 3–4 kg põõsa kohta

---

## Glen Dee

**Iseloomustus:**
- Hiline sort (valmib pärast 'Glen Ample')
- Väga magus, aromaatne
- Hea haiguskindlus
- **Ogadeta varred**

**Saagikus:** 2–3 kg põõsa kohta

**Valmimisaeg:** 'Glen Ample' järel → pikendab saagihooaega!

---

## Must aedpampel (Rubus occidentalis)

**Iseloomustus:**
- Mustad marjad, intensiivne maitse
- Väiksemad marjad kui punasel vaarikal
- Ogadega varred
- Kasvab kaartena – tipp juurdub, kui maad puudutab

**Eripärad:**
- Võib olla invasiivne (juurduvad tipud)
- Lõika maad puudutavad tipud ära, kui ei taha levikut

## Aastaringne hoolduskalender

### Varakevad (märts–aprill)

- [ ] Eemalda talvine kate (kui kasutasid)
- [ ] Seo oksad toe külge (kui lahti tulnud)
- [ ] Eemalda kahjustatud/surnud oksad
- [ ] Lühenda oksad toest kõrgemale (~ 15–20 cm üle toe)
- [ ] Väeta kompostiga (5–10 cm kiht juurte ümber)

### Kevad (mai)

- [ ] Harvendada uusi võrseid (jäta 6–8 tugevamat põõsa kohta)
- [ ] Kontrolli kahjureid (lehetäid, vaarikamardikas)
- [ ] Multši (hein, põhk, hakitud koor)

### Suvi – saagiaeg (juuli–august)

**Valmimise järjekord:**
```
Glen Ample → Must pampel → Glen Dee
(juuli)      (juuli keskp.)  (juuli lõpp–august)
```

- [ ] Korja iga 2–3 päeva tagant
- [ ] Korja hommikul, kui jahe
- [ ] Ära korja märgi (hallitavad)
- [ ] Külmuta või söö kohe (ei säili pikalt)

### Pärast saaki (august–september)

**Lõikus (OLULINE!):**
- [ ] Lõika viljunud oksad maapinna tasemele
  - Need on pruunikad, puitunud
  - Viljusid sel aastal, enam ei vilja
- [ ] Jäta uued rohelised võrsed
  - Need viljuvad järgmisel aastal
- [ ] Harvendada – jäta 6–8 tugevamat võrset põõsa kohta
- [ ] Seo uued võrsed toe külge

```
ENNE LÕIKUST:           PÄRAST LÕIKUST:
   |  |  |  |              |  |  |
   |  |  |  |              |  |  |
───┴──┴──┴──┴───        ───┴──┴──┴───
     viljunud +              ainult
     uued võrsed             uued võrsed
```

### Sügis (oktoober–november)

- [ ] Eemalda langenud lehed (haiguste ennetamine)
- [ ] Kontrolli tuge – kas on tugev talveks
- [ ] Multši juurte ümbrus (kaitse külma eest)
- [ ] Musta pampli puhul: lõika maad puudutavad tipud

### Talv

- Vaarikad on külmakindlad
- Tugev tuul + külm võib kahjustada – tuulekaitse kasulik
- Lumekatted on loomulik kaitse

Vaarikad vajavad tuge! Levinumad süsteemid:

    │        │        │   ← traat/nöör ülemine
    │   ┌────┼────┐   │
    │   │    │    │   │   ← traat/nöör alumine (50 cm)
    │   │    │    │   │
────┴───┴────┴────┴───┴────  maapind

           postid

- 2 traati: 60 cm ja 120 cm kõrgusel
- Oksad seotud traadi külge

## Kastmine

- Vaarikad armastavad **ühtlast niiskust**
- Eriti oluline marjade valmimise ajal
- Kuival perioodil: 2–3 korda nädalas
- Tilkkastmine on ideaalne
- Multš aitab niiskust hoida

## Kahjurid ja haigused

### Kahjurid

| Kahjur | Tunnused | Kahju | Tõrje |
|--------|----------|-------|-------|
| **Vaarikamardikas** | Marjad "ussitanud", väikesed valged tõugud sees | Rikutud marjad | Raputa hommikul ajalehele; püüdepurgid; sügisel mullakihi pööramine |
| **Lehetäid** | Keerdunud lehed, kleepuv kaste, sipelgad | Nõrgenenud võrsed, viirused | Seebivesi, neebiõli, tugev veejuga; lepatriinud |
| **Vaarikakoi** | Närbunud võrsetipud, augud pungades | Kaotatud võrsed ja saak | Lõika kahjustatud võrsed 10 cm terve koeni |
| **Vaarika-varreleedik** | Rõngakujulised turse-kohad vartel | Oksad murduvad | Lõika ja põleta nakatunud varred |
| **Ämbliklestad** | Kollakad/pronksjad lehed, peen võrk | Lehed varisevad, taim nõrgeneb | Pihusta vett lehtedele; väävlipreparaat |
| **Linnud** | Söödud marjad | Saagikadu | Kata võrguga |

### Haigused

| Haigus | Tunnused | Kahju | Ennetamine/tõrje |
|--------|----------|-------|------------------|
| **Hallhallitus** (Botrytis) | Hall ebemjas hallitus marjadel | Marjad söömiskõlbmatud | Hea õhuring, hõrenda, ära kasta marju |
| **Võrsehaigus** (Didymella) | Lillakad/pruunid laigud vartel, koor praguned | Võrsed surevad, saak väheneb | Eemalda ja põleta haigeid oksi; vaskpreparaat varakevadel |
| **Vaarikaroosmed** | Oranžid/kollased pustlid lehtedel | Lehed varisevad, taim nõrgeneb | Eemalda nakatunud lehed; resistentsed sordid |
| **Juuremädanik** (Phytophthora) | Taim närbub, juured pruunid/pehmed | Taim sureb | Paranda drenaaži; eemalda nakatunud taim; ära istuta samasse kohta |
| **Viirushaigused** | Kollased sooned lehtedel, kääbuskasv | Saak väheneb, taim degenereerub | Tõrje lehetäisid (viiruse kandjad); eemalda nakatunud taimed |

### Ennetamine on parim kaitse

1. **Hea õhuring** – hõrenda 6–8 võrseni põõsa kohta
2. **Kastmine mulda** – ära kasta ülevalt (seeneviljad levivad)
3. **Multšimine** – hoiab ära mullapritsmed lehtedele
4. **Hügieen** – eemalda haigeid oksi KOHE, põleta või prügikasti
5. **Külvikord** – ära istuta vaarikaid vana vaarika kohale
6. **Terve taim** – osta sertifitseeritud puukoolist

### Kahjurite ja haiguste seire

**Kontrolli regulaarselt (1× nädalas kevad–suvel):**
- [ ] Pungad ja võrsetipud – koi kahjustused
- [ ] Lehtede alumine pool – täid, lestad
- [ ] Varred – laigud, praod, turse
- [ ] Marjad – hallitus, tõugud

### Tõrjevahendite meelespea

| Probleem | Ökoloogiline lahendus | Keemiline (viimane variant) |
|----------|----------------------|----------------------------|
| Vaarikamardikas | Raputamine, püüdepurgid | Püretriinisprei (enne õitsemist) |
| Lehetäid | Seebivesi, neebiõli | Püretriinisprei |
| Hallhallitus | Hõrendamine, hügieen | Botrytiidi tõrjevahend |
| Võrsehaigus | Haigete okste eemaldamine | Vaskpreparaat varakevadel |

## Paljundamine

### Juurevõsudest (lihtne)

1. Kevadel või sügisel
2. Kaeva tugev juurevõsu välja
3. Istuta uude kohta
4. Kasta hästi

### Must pampel – tippude juurdumine

1. Painuta võrse tipp maha
2. Kaeva väike auk, suru tipp mulda
3. Kinnita kiviga
4. Juurdumine 4–6 nädalat
5. Lõika emapõõsast lahti, istuta ümber
