# Hooajaline tööde kalender

Rõngu, Lõuna-Eesti (külmavöönd ~6a)

## Rõngu kliima

| Parameeter | Tüüpiline kuupäev/väärtus |
|------------|---------------------------|
| **Viimane kevadine öökülm** | 20.–25. mai |
| **Ohutu istutusaeg õrnadele** | Pärast 1. juunit |
| **Esimene sügisene öökülm** | 20.–25. september |
| **Kasvuperiood** | ~130–140 päeva |
| **Külmavöönd** | 6a (-23°C kuni -20°C) |

⚠️ **NB:** Need on keskmised kuupäevad! Jälgi alati ilmateadet – öökülmad võivad tulla varem või hiljem.

**Praktiline reegel:**
- **Õrnad taimed välja** (tomat, kurk, basiilik): oota kuni 1. juuni
- **Poolkülmakindlad** (kartul, aeduba): mai keskpaik, kata vajadusel
- **Külmakindlad** (herned, sibul, porgand): aprill

## Märts

### Ettekasvatamine

| Nädal | Kultuur | Tegevus |
|-------|---------|---------|
| 1. nädal | **Kartul 'Solist'** | Eelidandamine (turbamullas, valge, <20°C) |
| 1.–2. nädal | **Porru** | Ettekasvatuse algus (külvikasti) |
| 2.–3. nädal | **Tomat** | Külv, 20–22°C |
| 2.–3. nädal | **Füüsal** | Külv (nagu tomat) |
| 3. nädal | Kartul | Tõsta jahedamasse, lase juurduda |
| 4. nädal | Tšilli | Ümberpotimine 9 cm pottidesse |

### Marjapõõsad

**Kuslapuud:**
- [ ] Vahede mõõtmine ja kontrollimine
- [ ] Lõikus enne pungade paisumist
- [ ] Surnud ja ristuvate okste eemaldamine
- [ ] Põõsa keskosa hõrendamine
- [ ] Väetamine kompostiga (10–15 L põõsa kohta)

**Vaarikad:**
- [ ] Talvise katte eemaldamine
- [ ] Okste sidumine toe külge
- [ ] Kahjustatud/surnud okste eemaldamine
- [ ] Okste lühendamine (15–20 cm üle toe)
- [ ] Väetamine kompostiga

**Sõstrad (must, punane, valge):**
- [ ] Lõikus enne pungade paisumist
- [ ] Must sõstar: eemalda 1/3 vanematest okstest (üle 3 a vanad)
- [ ] Punane/valge sõstar: lühenda külgharusid 2–3 pungani
- [ ] Hõrenda põõsaste keskosa
- [ ] Väetamine kompostiga (5–10 L põõsa kohta)

**Karusmari:**
- [ ] Lõikus enne pungade puhkemist
- [ ] Eemalda vanad oksad (üle 4 a), jäta 8–10 peaharu
- [ ] Hõrenda keskosa (jahukaste ennetamine!)
- [ ] Väetamine kompostiga

**Aroonia:**
- [ ] Vajadusel lõikus (minimaalne)
- [ ] Juurevõsude eemaldamine (kui ei soovi levikut)
- [ ] Väetamine kompostiga (3–5 L)

### Maasikad
- [ ] Vana multši eemaldamine
- [ ] Surnud ja kahjustatud lehtede eemaldamine
- [ ] Taimede seisukorra hindamine
- [ ] Vahede kontrollimine (30–35 cm taimede vahel)
- [ ] Liiga tihedalt kasvavate taimede eemaldamine
- [ ] Taimede kokkulugemine

### Õunapuud
- [ ] Lõikuse lõpetamine (enne pungade paisumist!)
- [ ] **Vaskpreparaat** (kärntõve ennetamiseks, pungade paisumisel)
- [ ] Väetamine kompostiga (20–30 L puu kohta)
- [ ] Mumifitseerunud viljade eemaldamine

### Kirsid ja ploomid
- [ ] Väetamine kompostiga (10–20 L puu kohta)
- [ ] Tüvekaitse eemaldamine
- [ ] **Vaskpreparaat** pungade paisumisel (monilioosi ennetamine)
- [ ] Kerged korrektiivlõikused (suurem lõikus suvel!)

### Kasvuhoone
- [ ] Ettevalmistamine ja koristamine
