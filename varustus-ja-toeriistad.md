# Varustus ja tööriistad

Kaevamisvaba (no-dig) aia tööriistad ja varustus.

---

## Põhitööriistad

### Igapäevased tööriistad

| Tööriist | Kasutus | Märkused |
|----------|---------|----------|
| **Aialabidas (käsilabidas)** | Istutamine, ümberistutamine | Kaevamisvabas aias peamine tööriist |
| **Hark** | Komposti ja multši liigutamine | 4 piiga, tugev |
| **Reha** | Pinna tasandamine, lehtede kogumine | Metallist või bambusest |
| **Oksakäärid** | Okste lõikamine, harvendamine | Teravad, bypass-tüüp parem |
| **Aiakäärid** | Suurem lõikus, oksad | Teritatud teraga |
| **Saag (oksasaag)** | Puude/põõsaste lõikus | Käi- või lavasaag |
| **Kastmisvoolik** | Kastmine | Piisava pikkusega |
| **Kastekann** | Väiksem kastmine, väetamine | 10 L sobiv |
| **Kühvel** | Komposti, multši laadimine | Tugev, ergonoomilise varrega |
| **Aiakäru** | Transport | Ühe rattaga hea manööverdamiseks |

### Spetsiaalsed tööriistad

| Tööriist | Kasutus | Märkused |
|----------|---------|----------|
| **Kõblas** | Umbrohu ja peenikeste juurte eemaldamine | Kasulik peenardel |
| **Istutuspulk** | Seemikute istutamine | Või teritatud kepp |
| **Markeerija** | Ridade märgistamine | Nöör + pulgad |
| **Mõõdulint** | Vahekaugused, peenrad | 5+ meetrit |
| **Mullasõel** | Komposti sõelumine | Väiksemad seemned |

## Lõiketööriistad (marjapõõsad ja puud)

| Tööriist | Kasutus | Hooldus |
|----------|---------|---------|
| **Oksakäärid (bypass)** | Elus oksad kuni 2 cm | Terita ja õlita regulaarselt |
| **Oksakäärid (anvil)** | Kuivanud oksad | Ei sobi elusale puidule |
| **Oksalõikur (pikavareline)** | Oksad 2–4 cm | Kõrged/paksud oksad |
| **Oksasaag** | Oksad 4+ cm | Puude pealõikus |
| **Redel (aiaredel)** | Puude hooldus | Kolmejalgne stabiilne |

### Lõiketööriistade hooldus

**Iga kasutuse järel:**
- Puhasta terast (eemalda vaik, mustus)
- Kuivata

**Regulaarselt:**
- Terita (terituskiviga või viilaga)
- Õlita liikuvad osad
- Desinfitseeri (70% alkohol) – eriti haigete okste lõikuse järel

## Kastmissüsteem

### Põhivarustus

| Vahend | Kasutus | Märkused |
|--------|---------|----------|
| **Voolik** | Kastmine | Vastupidav, painduv |
| **Voolikupidur** | Voolik korda | Hoiab vooliku korras |
| **Pritsnina** | Reguleeritav kastmine | Mitu režiimi |
| **Kastekann** | Täpsem kastmine, väetamine | 10–12 L |

### Tilkkastmine (soovituslik)

| Komponent | Otstarve | Märkused |
|-----------|----------|----------|
| **Peatoru** | Vee jaotus | 16 mm tavaline |
| **Tilgutid** | Vesi taime juurde | 2–4 l/h |
| **Ühendused** | T-ühendused, otsikud | Sama süsteemiga sobivad |
| **Filter** | Ummistuste vältimine | Puhasta regulaarselt |
| **Taimer** | Automaatne kastmine | Patareiga või veejõul |
| **Vähendaja** | Rõhu reguleerimine | Kui veerõhk liiga kõrge |

### Kastmiseks vajalik

**Kasvuhoonele:**
- Tilkkastmissüsteem (eelistatud)
- Veepaak sooja veega

**Peenardele:**
- Voolik + pritsnina
- Või tilkkastmine

## Hooajaline varustuse kontroll

### Kevad (märts)

- [ ] Kontrolli lõiketööriistade seisundit, terita
- [ ] Kontrolli kastmissüsteemi (lekked, ummistused)
- [ ] Valmista ette ettekasvatuse varustus
- [ ] Hangi multš ja kompost
- [ ] Kontrolli katteloori seisundit

### Suvi (juuni)

- [ ] Kontrolli tugesid ja nööre
- [ ] Paigalda linnuvõrgud
- [ ] Kontrolli tilkkastmist
- [ ] Hangi vajaminevad tõrjevahendid

### Sügis (september)

- [ ] Puhasta ja kuivata tööriistad
- [ ] Korista ettekasvatuse varustus
- [ ] Paigalda tüvekaitsed (puud)
- [ ] Valmista ette säilitusvarustus

### Talv (detsember)

- [ ] Hoiustatööriistade hooldus
- [ ] Tööriistade teritamine
- [ ] Järgmise hooaja planeerimine
- [ ] Tellimuste tegemine (seemned, väetised)
