# Porru kasvatamine

Porru (*Allium ampeloprasum* var. *porrum*) on lauguliste (Allium) hulka kuuluv kõrge külmataluvusega kultuur. Saak september–november, osaliselt mullas talvitub.

---

## Ülevaade

| Parameeter | Väärtus |
|------------|---------|
| Asukoht 2026 | Peenar 3 (koos sibula, porgandiga) |
| Ettekasvatamine | Märts (sees) |
| Istutamine | Mai keskpaik–lõpp |
| Saak | September–november (kuni külmadeni) |
| Kasvuaeg | ~120–150 päeva |
| Külmataluvus | Kuni -10°C (mõned sordid veelgi) |

---

| **'Carentan'** ⭐ | Hiline | Väga hea | Talvikindel, jäme varr |
| **'Musselburgh'** ⭐ | Hiline | Väga hea | Šoti klassika, jõuline |

**Soovitus:** 'Carentan' või 'Musselburgh' – jäävad mulda hilissügiseni.

## Ettekasvatamine (märts)

Porru vajab pika kasvuaja tõttu **ettekasvatamist** märtsis.

| Tegur | Juhis |
|-------|-------|
| Alustamine | Märts algus |
| Temperatuur | 15–20°C |
| Sügavus | 1 cm |
| Idanemine | 10–14 päeva |
| Kasvuaeg sees | 10–12 nädalat (kuni ~15–20 cm kõrgused) |
| Karastamine | Mai algus – vii päeval õue |
| Istutamine | Mai keskpaik–lõpp |

```
KÜLVIKAST (märts):
┌──────────────────────────────────────────┐
│  │  │  │  │  │  │  │  │  │  │  │  │  │  │
│  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  ↑  │
│  Porru istikud (5 cm vahe reas)          │
│  Kasvavad 10–12 nädalat siseruumis       │
└──────────────────────────────────────────┘
```

## Istutamine

### Istutamistehnika

Porru istutamisel kasutatakse **süvaistutust** – see moodustab pika valge osa.

| Tegur | Juhis |
|-------|-------|
| Aeg | Mai keskpaik–lõpp (pärast öökülmade ohtu) |
| Augu sügavus | 15 cm |
| Vahe reas | 15 cm |
| Ridade vahe | 30 cm |

**Sammud:**
1. Tee kepiga 15 cm sügavused augud
2. Lase istik auku (ära täida mullaga!)
3. **Kasta auk veega täis** – muld settib ise istiku ümber
4. Ära tihedalt suru

```
ISTUTAMINE:
         ↓ istik auku
    ─────┼──────────────────
         │
         │  ← 15 cm sügav auk
         │     (muld settib ise)
         │
    ─────┴──────────────────
         (põhi)
```

**Miks nii?** Pikem valge osa (blanšeeritud) = pehme, mahe maitse. Süvaistutus tekitab valge osa ilma muldamiseta.

## Hooldus

### Muldamine (valikuline lisatoodang)

Kui soovid **pikemaks valgeks osaks**, mulda kasvuperioodil:
- Tõsta mulda/komposti varre ümber 2–3 korda suvel
- Iga kord lisa ~5 cm kiht
- Katta osa, mitte lehti

### Kastmine

| Periood | Sagedus | Märkused |
|---------|---------|----------|
| Istutamise järel | Regulaarselt | Kuni taimed kinni juurduvad |
| Kasvuperiood | 1× nädalas kuival | Ühtlane niiskus |
| Sügis | Harvem | Vähenenud veevajadus |

### Väetamine

| Aeg | Väetis | Märkused |
|-----|--------|----------|
| Istutamisel | Kompost auku | Lämmastikurikas algus |
| Juuni–juuli | Nõgesevesi 1:10 | Kiirendab kasvu |
| Augustis | Ei vajalik | Ettevalmistus talveks |

## Koristamine

### Millal korjata

| Märk | Kirjeldus |
|------|-----------|
| Aeg | September – kuni esimeste tugevate külmadeni |
| Varr | Vähemalt 2–3 cm läbimõõduga |
| Paindlikkus | Korja vastavalt vajadusele – ei pea korraga |

**Porru eripära:** Ei pea korraga koristama! Jäta mulda ja korja jooksevalt kuni novembrini (mõned sordid talvituvad mullas).

### Koristamine

1. Tõmba varrest või kaeva hargiga
2. Lõika juured lühidaks
3. Eemalda välimised kuivanud lehed
4. ÄRA pese enne kasutamist

### Talves mullas

**Talvekindlad sordid** ('Carentan', 'Musselburgh') saab jätta mulda:
- Kata multšiga (10–15 cm)
- Korja vastavalt vajadusele kuni novembrini
- Külm parandab maitset (suhkrusisaldus kasvab)!

## Aastaringne kalender

| Kuu | Tegevus |
|-----|---------|
| **Märts** | **Ettekasvatamise algus** (külvikast, 1 cm sügav) |
| Aprill | Karastamine, vajadusel ümberistutus suurematesse pottidesse |
| Mai algus | Karastamine õues (päeval) |
| **Mai keskpaik–lõpp** | **Istutamine õue** (15 cm sügavad augud) |
| Juuni | Muldamine (pikema valge osa jaoks) |
| Juuni–august | Kastmine, väetamine (nõgesevesi) |
| September | **Koristamise algus** |
| September–november | Jooksev korjamine mullas |
| November | Viimane koristamine enne tugevaid külmi |
