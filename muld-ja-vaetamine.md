# Muld ja väetamine

Kaevamisvaba (no-dig) aia mulla hooldus ja väetamine.

---

## Kaevamisvaba aia põhimõtted

### Miks mitte kaevata?

| Kaevamine | Kaevamisvaba |
|-----------|--------------|
| Hävitab mulla struktuuri | Säilitab mulla struktuuri |
| Tapab kasulikud organismid | Toidab mullaelustikku |
| Toob umbrohuseemned pinnale | Umbrohuseemned jäävad sügavale |
| Vajab igal aastal kordamist | Muld paraneb iga aastaga |
| Palju tööd | Vähem tööd |

### Kaevamisvaba aia loomine

**Uue peenra rajamine:**

1. **Niida/talluta olemasolev taimestik** maha
2. **Kata papp/kartong** (kattub 10–15 cm) – summutab umbrohtu
3. **Lisa 15–20 cm komposti** peale
4. **Oota 2–4 nädalat** või istuta kohe (suuremad taimed)
5. **Multši** 5–10 cm

**Olemasoleva peenra hooldus (igal aastal):**

1. **Lisa 5–10 cm komposti** varakevadel (märts–aprill)
2. **Multši** pärast istutamist
3. **Ära kaeva!** – lase vihmaussidel tööd teha

## Kompost

Kompost on kaevamisvaba aia alus – see on nii väetis kui mullaparandaja.

### Hea komposti tunnused

| Omadus | Kirjeldus |
|--------|-----------|
| Värv | Tumepruun kuni must |
| Lõhn | Meeldiv, mullane (mitte mädane!) |
| Tekstuur | Murenev, ühtlane |
| Temperatuur | Jahtunud (toatemperatuur) |
| Struktuur | Algmaterjalid ei ole äratuntavad |

### Kompostimise alused

**Roheline materjal (lämmastik, N):**
- Värsked taimed, muru
- Köögiviljajäätmed
- Kohvipaks, teelehed
- Nõgese-, heinakoorem

**Pruun materjal (süsinik, C):**
- Kuivanud lehed
- Põhk, hein
- Papp, paber (värvimata)
- Oksapuru, saepuru

**Suhe:** ~3 osa pruuni : 1 osa rohelist (mahuliselt)

### Kompostihunnik vs kompostikast

| Meetod | Eelised | Puudused |
|--------|---------|----------|
| **Kuumkompost** (aktiivne) | Kiire (2–3 kuud), tapab umbrohuseemned | Vajab pööramist, jälgimist |
| **Külmkompost** (passiivne) | Lihtne, vähe tööd | Aeglane (6–12 kuud), seemneid võib säilida |
| **Kompostikast** | Korras, kompaktne | Väiksem maht |
| **Hunnik** | Suur maht, odav | Võtab ruumi, laialivalguv |

### Kuumkomposti tegemine (kiire meetod)

**Sammud:**
1. **Kogu materjali** vähemalt 1 m³ (kriitilise massi jaoks)
2. **Sega rohelised ja pruunid** korraga kokku
3. **Niisuta** – nagu välja pigistatud käsn
4. **Jälgi temperatuuri** – peaks tõusma 55–65°C-ni
5. **Pööra** kui temp langeb (iga 1–2 nädala tagant)
6. **Valmis** 2–3 kuuga

**⚠️ Oluline:** Kuumkompost (>55°C) tapab umbrohuseemned ja haigustekitajad!

### Mida MITTE kompostida

| Väldi | Põhjus |
|-------|--------|
| Haigeid taimi | Haigused võivad säilida |
| Seemneid kandvat umbrohtu | Seemned võivad säilida |
| Liha, kala, piimatooted | Lõhn, kahjurid |
| Koera-/kassiväljaheidet | Parasiidid |
| Ravimtaimedega töödeldud taimi | Jäägid võivad säilida |
| Rasvast, õli | Ei lagune hästi |

### Kompostikogused peenrale

| Kasutus | Kogus | Sagedus |
|---------|-------|---------|
| Uus peenar | 15–20 cm kiht | Ühekordne |
| Olemasolev peenar | 5–10 cm kiht | Igal kevadel |
| Puude/põõsaste väetamine | 10–20 L taime kohta | Igal kevadel |
| Kasvuhoone | 5–10 cm kiht | Igal kevadel |

**Ligikaudne vajadus 4 peenrale (1 × 9 m):**
- Uus aia rajamine: ~3–4 m³ komposti
- Iga-aastane hooldus: ~1,5–2 m³ komposti

## Vermikomposti (ussikompost)

Vermikomposti on kompostiusside abil valmistatud ülimalt kvaliteetne orgaaniline väetis. Ussid töötlevad orgaanilisi jäätmeid ja toodavad biohuumust, mis on tavalisest kompostist 5–10 korda toitainerikkam.

### Miks vermikomposti?

| Eelis | Selgitus |
|-------|----------|
| Kõrge toitaineväärtus | Sisaldab rohkem N, P, K kui tavaline kompost |
| Mikroobiderohkus | Sisaldab kasulikke baktereid ja seeni |
| Kasvustimulaator | Sisaldab humiinhappeid ja hormoone |
| Ei põleta taimi | Võib kasutada lahjendamata |
| Kompaktne süsteem | Sobib väikesesse kohta, ka tuppa |
| Aastaringne | Töötab talvelgi (toas) |

### Kompostiussi liigid

| Liik | Ladina nimi | Sobivus | Märkused |
|------|-------------|---------|----------|
| **Punane kompostiuss** ⭐ | Eisenia fetida | Parim! | Kiire, toitainerohke, talub laias tempvahemikus |
| **Teine kompostiuss** | Eisenia andrei | Väga hea | Sarnane punasega, natuke väiksem |
| **Aafrika ööliblikas** | Eudrilus eugeniae | Soe kliima | Ei sobi Eestisse (vajab >15°C) |

**⚠️ Tavaline vihmauss (Lumbricus terrestris) EI SOBI kompostimiseks!** Ta kaevab sügavale mulda ega toitu pinnasest.

### Ussikasti ehitamine

**Lihtne DIY ussikast:**

```
┌─────────────────────────────┐
│  KAAS (pimedaks + niiskeks) │
├─────────────────────────────┤
│  TOITMISE KIHT              │  ← Köögiviljajäätmed
│  ~5 cm                       │
├─────────────────────────────┤
│  USSID + ALLAPANU           │  ← Ussid + niiske papp/turvas
│  ~15 cm                      │
├─────────────────────────────┤
│  DRENAAŽ (augud põhjas)     │  ← Vedelik välja
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ │
└─────────────────────────────┘
     ↓ Ussi-"tee" (vedelik)
```

**Materjalid:**
- 2 plastikkasti (60–80 L), üksteise sees
- Alumisel kastil augud põhjas (drenaaž)
- Kaas (pimedus on oluline!)
- Kraanike alumises kastis (valikuline)

**Allapanu:**
- Rebitud papp / kartong (niisutatud)
- Kookoskiud
- Turvas (väike kogus)
- Lehtedekõdu

### Söötmine

**Mida ANDA:**

| Materjal | Kogus | Märkused |
|----------|-------|----------|
| Köögiviljajäätmed | Peamine | Koorid, lehed, varred |
| Puuviljajäätmed | Mõõdukalt | Banaanikoored, õunad |
| Kohvipaks | Hea! | Ussidele meeldib, neutrahne pH |
| Teepuru | Jah | Koos filtriga |
| Munakoored | Jah | Purusta peeneks (kaltsium) |
| Papp, paber | Jah | Rebitud, niisutatud ("pruun" materjal) |
| Surnud lehed | Jah | Mõõdukalt |

**Mida MITTE anda:**

| Materjal | Põhjus |
|----------|--------|
| Liha, kala, luu | Hais, kahjurid |
| Piim, juust | Hais, hallitus |
| Rasv, õli | Ei lagune |
| Küüslauk, sibul | Ussid ei talu |
| Tsitrusvilju (palju) | Liiga happeline |
| Teraviljatooteid | Lähevad hallitama |
| Haigeid taimi | Haigused võivad säilida |

### Hooldus

**Optimaalsed tingimused:**

| Parameeter | Väärtus |
|------------|---------|
| Temperatuur | 15–25°C (parim 20°C) |
| Niiskus | Nagu pigistatud käsn |
| pH | 6–7 (neutraalne) |
| Valgus | Pime (ussid pelgavad valgust) |
| Õhk | Hea ventilatsioon |

**Igapäevane hooldus:**
- Söötmine 2–3× nädalas
- Ära anna rohkem kui ussid ära söövad (~nende kaal päevas)
- Sega aeg-ajalt (iga 1–2 nädalat)

**Probleemid ja lahendused:**

| Probleem | Põhjus | Lahendus |
|----------|--------|----------|
| Hais | Liiga palju toitu, liiga märg | Vähenda söötmist, lisa papp |
| Kärbsed | Liiga palju puuvilju, lahtine toit | Kata toit paberi/papiga |
| Ussid põgenevad | Liiga niiske/kuiv/happeline | Korrigeeri tingimusi |
| Hallitus | Liiga palju tsitrust, halb ventilatsioon | Lisa lupja, tuuluta |
| Ussid ei söö | Liiga külm või toit pole sobiv | Soojenda, vaheta toitu |

### Vermikomposti koristamine

**Meetod 1: Valguse meetod (lihtne)**
1. Lükka valmis kompost kasti ühte otsa
2. Lisa värsket toitu teise otsa
3. Oota 1–2 nädalat (ussid kolivad toidu juurde)
4. Korista valmis kompost

**Meetod 2: Sõelumine**
1. Tõsta sisu välja
2. Sõelu läbi (5 mm sõel)
3. Ussid ja suuremad tükid tagasi kasti

**Saak:**
- ~1/3 algmaterjali mahust
- Algul vähem, kogenumal kasutajal rohkem

### Vermikomposti kasutamine

**Tahkena:**

| Kasutus | Kogus | Kuidas |
|---------|-------|--------|
| Istutusauk | 1–2 peotäit | Sega mullaga |
| Seemikutele | 10–20% | Sega potimullaga |
| Peenrale | 0,5–1 cm kiht | Pealekülv |
| Taimedele | 1 peotäis | Juurte ümber |
| Murule | Õhuke kiht | Üle muru |

**Vermikomposti tee (vedel väetis):**

1. **Leotamine:** 1 osa vermikomposti : 5 osa vett
2. **Aeg:** 24–48 tundi (sega aeg-ajalt)
3. **Kurna:** Läbi riide või sõela
4. **Kasuta:** Lahjendamata või 1:5
5. **Kasta:** Juurte ümber või pihusta lehtedele

**⭐ Vermikomposti tee on suurepärane kasvustimulaator!**
- Sisaldab kasulikke mikroobe
- Aitab taimedel toitaineid omastada
- Tõstab haiguskindlust

### Ussikasti käivitamine

**1. Valmista kast ette**
- Puurida augud (drenaaž + ventilatsioon)
- Lisa alumine kast (vedeliku kogumiseks)

**2. Lisa allapanu (~15 cm)**
- Rebitud, niisutatud papp/kartong
- Kookoskiud või turvas
- Natuke mulda (mikroobid)

**3. Lisa ussid**
- Alusta ~500 g ussidega (~500–1000 tk)
- Laota allapanu peale
- Kata paberi/papiga

**4. Oota 3–5 päeva**
- Lase ussidel kohaneda
- Ära söötmist kohe

**5. Alusta söötmist**
- Esimene nädal: vähe
- Järk-järgult suurenda
- Ussid kahekordistuvad ~3 kuuga

### Kust osta kompostiusse

| Allikas | Hind | Märkused |
|---------|------|----------|
| Teised ussikompostijad | Sageli tasuta | Parima kohanemisega |
| Kalapoed (sööt) | 3–5 € tops | Punased ussid, väike kogus |
| Aianduspoed | 10–20 € | Vermikomposti starterkit |
| E-poed | 15–30 € / 500 g | Spetsiaalselt kasvatatud |

**⭐ Soovitus:** Alusta väiksema kogusega – ussid paljunevad kiiresti!

### Vermikomposti aastaplaan

| Kuu | Tegevus |
|-----|---------|
| Kevad | Suurenda söötmist (ussid aktiivsemad) |
| Suvi | Jälgi temperatuuri! Vii jahedamasse kohta kui >25°C |
| Sügis | Korista kompost, valmista talveks |
| Talv | Hoia toas (>10°C), vähenda söötmist |

## Vedelvätised

### Nõgesevesi (üldväetis)

Nõgesevesi on parim universaalne vedelväetis – sisaldab lämmastikku, kaaliumi ja mikroelemente.

**Valmistamine:**

1. **Kogu nõgeseid** (enne õitsemist parim)
2. **Täida ämber/tünn** 1/3–1/2 ulatuses nõgestega
3. **Lisa vett** peale
4. **Kata** (vähendab lõhna)
5. **Sega** iga paari päeva tagant
6. **Valmis** 2–3 nädalaga (lõpetab käärimise, ei mullita enam)

**Kasutamine:**

| Lahjendus | Kasutus | Sagedus |
|-----------|---------|---------|
| 1:10 (vesi:nõgesevesi) | Kastmine juurte ümber | Iga 1–2 nädala tagant |
| 1:20 | Leheväetis (pihustamine) | Iga 2–3 nädala tagant |

**Sobib:**
- Tomatid, kurgid, kõrvitsalised (suured toitjaained)
- Kapsas, lehtköögiviljad
- Marjapõõsad

**⚠️ Ära kasuta:**
- Kaunviljadele (herned, oad) – neil on oma lämmastik
- Sibulale, küüslaugule – eelistavad vähem lämmastikku
- Juurviljadele (liiga palju N = lehti, vähe juurt)

---

### Sõnnikuvesi (lämmastikurikas)

**Valmistamine:**
1. Lisa 1 osa kõdunud sõnnikut : 10 osa vett
2. Lase seista 1–2 nädalat
3. Sega aeg-ajalt

**Kasutamine:**
- Lahjenda 1:5 enne kasutamist
- Kasta juurte ümber
- Kasuta kasvuperioodil (mai–juuli)

**⚠️ Oluline:**
- Kasuta AINULT kõdunud sõnnikut (värske põletab juuri)
- Ära kasuta hobuse/lehmasõnnikut värskena
- Kanasõnnik on tugevaim – lahjenda rohkem (1:20)

---

### Taimeleotised (orgaanilised tõrjevahendid)

**Küüslauguleotis (kahjuritõrje):**
1. Purusta 4–5 küüslauguküünt
2. Leota 1 L vees 24 tundi
3. Kurna, lisa tilk nõudepesuvahendit
4. Pihusta lehtedele (ka alt!)
5. Korda iga 3–5 päeva

**Kõrvitsaleotis (jahukaste vastu):**
1. 1 L piima + 9 L vett
2. Pihusta lehtedele
3. Korda iga nädal

**Sooda-seebi lahus (seenhaigused):**
1. 1 tl söögisooda + tilk vedelat seepi
2. 1 L vett
3. Pihusta lehtedele

## Tahked väetised

### Orgaanilised väetised

| Väetis | NPK | Kasutus | Kogus |
|--------|-----|---------|-------|
| **Kompost** | ~1-1-1 | Universaalne, mullaparandaja | 5–10 cm kiht |
| **Kõdunud sõnnik** | ~2-1-2 | Lämmastikurikas, mullaparandaja | 3–5 cm kiht |
| **Luujahu** | 3-15-0 | Fosfor (juured, õied) | 50–100 g/m² |
| **Puutuhk** | 0-1-5 | Kaalium (viljad, talvekindlus) | 100–200 g/m² |
| **Sarvijahu** | 12-0-0 | Aeglane lämmastik | 50–100 g/m² |
| **Merevetikajahu** | 1-0-2 | Mikroelemendid | 50 g/m² |
| **Vermikomposti tee** | – | Mikroobid, kasvustimulaator | Lahjendatud |

### NPK seletus

| Element | Sümbol | Roll | Puuduse märgid |
|---------|--------|------|----------------|
| **Lämmastik** | N | Lehekasv, roheline värv | Kahvatud, kollased lehed |
| **Fosfor** | P | Juured, õied, viljad | Lillakad lehed, nõrk kasv |
| **Kaalium** | K | Viljad, haiguskindlus, talvekindlus | Leheservade pruunistumine |

### Puutuhk

Puutuhk on suurepärane kaaliumiallikas ja tõstab mulla pH-d.

**Kasutamine:**
- **Kogus:** 100–200 g/m² (peotäis põõsa kohta)
- **Aeg:** Kevad või sügis
- **Sobib:** Tomatid, kartul, puuviljad, marjad, sibul
- **Väldi:** Mustikatele, rododendronitele (happelembsed)

**⚠️ Ära kasuta:**
- Briketitest, töödeldud puidust tuhka
- Grillsöetuhka (lisaained)
- Liiga suures koguses (tõstab pH liiga kõrgeks)

### Luujahu

Aeglaselt toimiv fosforväetis – hea juurte arengule ja õitsemisele.

**Kasutamine:**
- **Kogus:** 50–100 g/m²
- **Aeg:** Kevad (istutamisel) või sügis
- **Sobib:** Sibul, küüslauk, puud, põõsad, sibullilled

### Munakoored

Kaltsiumiallikas, aitab vältida tomatite lõheviljalisust.

**Kasutamine:**
1. Kuivata ja purusta peeneks
2. Lisa istutusauku (tomat, paprika)
3. Või sega komposti hulka

## Mulla pH

### Eesti muldade pH

Enamik Eesti muldi on **kergelt happelised** (pH 5,5–6,5), mis sobib enamikele kultuuridele.

**Optimaalsed pH väärtused:**

| pH vahemik | Kultuurid |
|------------|-----------|
| 5,0–5,5 | Mustikas, jõhvikas, rododendron |
| 5,5–6,5 | Kartul, tomat, porgand, kapsas |
| 6,0–7,0 | Enamik köögivilju, marjapõõsad |
| 6,5–7,5 | Sibul, küüslauk, spinat, kaunviljad |

### pH mõõtmine

**Lihtne meetod:**
- Osta pH-tester (aianduskeskusest, ~10–20 €)
- Mõõda kevadel enne väetamist

**Laboris:**
- Täpsem analüüs: Põllumajandusuuringute Keskus
- Saad teada ka toitainete sisalduse

### pH korrigeerimine

**Liiga happeline muld (pH < 6,0):**
- Lisa **dolomiitjahu** või **lupja**
- Kogus: 100–200 g/m²
- Aeg: Sügis (reageerib aeglaselt)
- Puutuhk tõstab samuti pH-d

**Liiga leeliseline muld (pH > 7,5):**
- Lisa **turvast** või **okaspuuokste hakkepuitu**
- Väävli lisamine (aiandusväävli)
- Happeline kompost (okaspuunõeltega)

| **Punane ristik** | Liblikõieline | Sügav juurestik, N | Kevad–suvi |
| **Faatseelia** | Õistaimeline | Kiire kasv, mesilaste toit | Kevad–suvi |
| **Sinep** | Ristõieline | Kiire kasv, biotõrje | Kevad–sügis |
| **Tatrar** | Tatralised | Kiire, toob fosforit pinnale | Suvi |

### Haljasväetise kasutamine

**Sügisene külv (pärast saaki):**
1. Kohe pärast viimast saaki
2. Külva talirukis, talinäär, või ristik
3. Lase kasvada talve üle
4. Kevadel: niida maha, kata kompostiga

**Suvine vahekultuur:**
1. Tühja kohta (enne hilisemat istutust)
2. Külva faatseelia või sinep
3. Niida enne õitsemist (ei seemneks)
4. Jäta pinnale multšiks

### Liblikõieliste eriline roll

Liblikõielised (herned, oad, ristik, näär) **seovad õhulämmastikku** juuremügarate abil:
- Pärast saaki jäta juured mulda (lämmastik vabaneb)
- Järgmisel aastal istuta lämmastiku-lembesed kultuurid (tomat, kapsas, kurk)
